# 例文ID: 日常-3.1

## 原文
vtʃi kino ʃehrid ætatdo çiti mataw, kar ɔki hn̩o jomtade çit ara.

## 現代日本語訳
私は昨日町で会った人を待っています、彼は大きな本を読んでいた人です。

## 形態素解析
vtʃi = 私は（代名詞、1人称単数、主格）
kino = 昨日（時間副詞）
ʃehrid = 町で（名詞、無生、前舌、所格単数不定形）
ætatado = 会った（動詞、前舌、過去、関係節標識）
çiti = 人を（名詞、有生、前舌、対格単数不定形）
mataw = 待っています（動詞、後舌、現在、1人称単数）
kar = 彼は（代名詞、3人称単数、主格）
ɔki = 大きい（形容詞、後舌、前舌名詞修飾、対格単数不定形）
hn̩o = 本を（名詞、無生、後舌、対格単数不定形）
jomtade = 読んでいた（動詞、後舌、過去、関係節標識）
çit = 人（名詞、有生、前舌、語幹）
ara = です（コピュラ、現在）

## 使用単語リスト
- vtʃ: 代名詞、1人称単数
- kino: 副詞、時間、「昨日」
- ʃehr: 名詞、無生、前舌、「町」
- æta: 動詞、前舌、「会う」
- çit: 名詞、有生、前舌、「人」
- mat: 動詞、後舌、「待つ」
- kar: 代名詞、3人称単数、「彼」
- ɔk: 形容詞、後舌、「大きい」
- hn̩: 名詞、無生、後舌、「本」
- jom: 動詞、後舌、「読む」
- ar: 動詞、不規則、「ある/いる」（コピュラとしての使用）

## 文法注釈
この例文は二つの関係節を含む複雑な構造を示しています：

1. 第一の関係節: 「kino ʃehrid ætatdo」（昨日町で会った）
   - 関係節標識「-do」（前舌動詞用）が動詞「æta」（会う）の過去形に付加されています
   - この関係節は主節の目的語「çiti」（人を）を修飾しています

2. 第二の関係節: 「ɔki hn̩o jomtade」（大きな本を読んでいた）
   - 関係節標識「-de」（後舌動詞用）が動詞「jom」（読む）の過去形に付加されています
   - この関係節は文末の「çit」（人）を修飾しています

また、形容詞「ɔk」（大きい）が後舌形容詞でありながら、前舌名詞「hn̩」（本）を修飾する際の母音調和にも注目してください。

文全体は、コンマで区切られた二つの節からなり、後半はコピュラ「ara」（です）を使った同定文となっています。
