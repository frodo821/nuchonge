# ニンゴト語ブックレット(第3版)
## 音韻体系から会話まで

---

## 序論：時を超える日本語

本書は、現代日本語から1000年後に発達する可能性のある言語体系、ニンゴト語を言語学的観点から考察したものです。1000年という長い時間の中で、日本語は膠着語から孤立語を経て屈折語へと変化すると予測されます。本書を通じて、この未来の日本語の音韻体系、形態変化、統語構造を学び、基本的な会話ができるようになることを目指します。

### 本書の使用上の注意

本書では、教育的目的で形態素境界を「-」記号で示していますが、実際のニンゴト語表記では、これらのハイフンは使用されません。すべての語は実際には連続して表記されます。たとえば、「vtʃi-ne nam-as」は「vtʃine namas」と表記されます。

## 第1部：音韻体系の変化

### 1.1 音韻変化のタイムライン

現代日本語の音韻体系は1000年の間に以下のような変化を遂げます：

| 時期       | 主要な音韻変化       | 例示                                  |
| ---------- | -------------------- | ------------------------------------- |
| 第1-2世紀  | 語末母音の弱化と脱落 | /jama/ → /jam/ (山)                   |
| 第3-4世紀  | 子音クラスタの形成   | /sakura/ → /sakr/ → /skr/ (桜)        |
| 第5-6世紀  | 母音調和の発達       | /taberu/ → /taber/ → /tabar/ (食べる) |
| 第7-8世紀  | 音素の合流と分岐     | /hito/ → /hit/ → /çit/ (人)           |
| 第9-10世紀 | 音韻体系の再編成     | /mai/ → /mai/ → /mæ/ (前)             |

### 1.2 ニンゴト語の音韻体系

#### 1.2.1 母音体系

| 母音     | 前舌 | 中舌 | 後舌 |
| -------- | ---- | ---- | ---- |
| 狭母音   | i    |      | u    |
| 中母音   | e    | ə    | o    |
| 半広母音 | æ    |      | ɔ    |
| 広母音   |      | a    |      |

#### 1.2.2 子音体系

| 調音位置→<br>調音方法↓ | 両唇音 | 唇歯音 | 歯茎音 | 後部歯茎音 | 硬口蓋音 | 軟口蓋音 | 声門音 |
| ---------------------- | ------ | ------ | ------ | ---------- | -------- | -------- | ------ |
| 破裂音                 | p, b   |        | t, d   |            |          | k, g     |        |
| 摩擦音                 |        | f, v   | s, z   | ʃ, ʒ       | ç        |          | x      |
| 鼻音                   | m      |        | n      |            |          | ŋ        |        |
| 流音                   |        |        | r, l   |            |          |          |        |
| 接近音                 | w      |        |        |            | j        |          |        |

#### 1.2.3 音節構造

基本的な音節の構造は (C)(C)(C)V(C)(C) であり、最大音節構造 CCCVCC までが許容されます。
また、許容される最大の子音クラスタである CCC は、構成する子音のいずれかが聞こえ度の高い子音である場合に限られ、例えば /çts/ などの聞こえ度の低い子音のみで構成されたクラスタは許容されません。

一般的に有声音は聞こえ度が比較的高く、無声音は比較的低いです。vやjなどの聞こえ度の高い子音は、比較的大きな子音クラスタを形成する傾向にあります。

また、/r̃/, /l̃/, /ñ/, /m̃/ の4つの子音は、音節主音としての機能を持ち、母音の代わりに音節核となることができます。つまり、これらの子音を音節核とする /kr̃s/ のような音節が許容されます。通常 /ñ/, /m̃/ は前舌母音、/r̃/, /l̃/ は後舌母音として扱われます。

### 1.3 子音クラスタの対処戦略

日本語が屈折語化する過程で発生する複雑な子音クラスタに対して、以下の対処戦略が発達します：

#### 1.3.1 母音挿入
- **前舌母音環境**：「i」の挿入
  例：「木から」→ 「k kr」→ 「k ikr」

- **後舌母音環境**：「a」や「u」の挿入
  例：「山から」→ 「jam kr」→ 「jam akr」

#### 1.3.2 共鳴音の音節主音化
- **鼻音**：n, m → ñ, m̃
  例：「本に」→ 「hon n」→ 「hñ ñ」

- **流音**：r, l → r̃, l̃
  例：「山らから」→ 「jam r kr」→ 「jam r̃ kr̃」

#### 1.3.3 子音の同化と簡略化
- **調音点の同化**：
  例：「日本語」→ 「nihongo」→ 「nihoŋgo」

- **子音脱落**：
  例：「行きました」→ 「ik mast」→ 「ikast」

### 1.4 綴りと発音の関係
ニンゴト語では、ほぼ綴りのとおりに発音しますが、一部の例外があります。
特に子音クラスタでは、以下の綴りについて子音が発音困難ないしは発音不可能のため、黙字化することで発音が簡略化されました。

- ñn あるいは ñm → ñ
- m̃m あるいは m̃n → m̃

単語によっては一見すると最大音節構造が許容する音素数を超える音節が存在するように見えますが、このような処理によって正当化されます。

## 第2部：形態統語論的変化（膠着語→孤立語→屈折語）

### 2.1 孤立語化プロセス（第1-2世紀）

最初の変化として、膠着語である現代日本語が孤立語化します。助詞が独立し、動詞活用が単純化します。

#### 2.1.1 格助詞の弱化
* 「が」→「g」: 「山**が**見える」→「山 **g** 見える」
* 「を」→「o」: 「本**を**読む」→「本 **o** 読む」 
* 「に」→「n」: 「学校**に**行く」→「学校 **n** 行く」
* 「で」→「d」: 「公園**で**遊ぶ」→「公園 **d** 遊ぶ」

#### 2.1.2 動詞活用の単純化
* 「書**く**」「書**き**」「書**け**」→「**kak**」(語幹のみ)
* 時制・アスペクト標識が独立要素化：「読み **ta**」（読んだ）

### 2.2 屈折語化プロセス（第3-10世紀）

#### 2.2.1 後置詞の接尾辞化
* 「wat n」→「wat-ñ」（私に）
* 「hon o」→「hon-o」（本を）

#### 2.2.2 代名詞と後置詞の融合
* 「wat」（私）：主格
* 「wat-ñ」（私に）：与格
* 「wat-o」（私を）：対格

#### 2.2.3 動詞と時制・相・法標識の融合
* 「読み」＋「ta」（過去）→「yom-ta」
* 「行き」＋「ro」（未来）→「ik-ro」

## 第3部：名詞の曲用体系

### 3.1 名詞分類

ニンゴト語では、名詞は以下の基準で分類されます：

1. **母音調和による分類**
   - 前舌母音系名詞（i, e, æを含む語幹）
   - 後舌母音系名詞（a, o, uを含む語幹）

2. **文法性による分類**
   - 有生名詞
   - 無生名詞

3. **数による分類**
   - 単数
   - 複数（「ら」由来の「r」接尾辞が発達）

4. **定性による分類**
   - 不定
   - 定（「s」接尾辞が発達）

### 3.2 格体系

ニンゴト語は8つの基本格を持ちます：

| 格   | 機能             | 由来                    |
| ---- | ---------------- | ----------------------- |
| 主格 | 動作・状態の主体 | 「が」→「g」→「i/a」    |
| 対格 | 動作の対象       | 「を」→「o」→「e/o」    |
| 与格 | 間接目的語       | 「に」→「n」→「ñ」      |
| 所格 | 場所・手段       | 「で」→「d」→「d」      |
| 属格 | 所有・関係       | 「の」→「no」→「ne/no」 |
| 出格 | 起点・比較       | 「から」→「kr」→「kr̃」  |
| 向格 | 方向・目標       | 「へ」→「h」→「he/h」   |
| 共格 | 共同・相互関係   | 「と」→「t」→「t/to」   |

厳密には呼格も存在しますが、呼格は主格と同じ形を持ち、特に区別を受けません。

#### 3.2.1. 主格
主格は動作や状態の主体を示します。現代日本語の「が」や「は」に相当します。

#### 3.2.2 対格
対格は動作の対象を示します。現代日本語の「を」に相当します。

#### 3.2.3 与格
与格は間接目的語を示します。現代日本語の「に」に相当します。

#### 3.2.4 所格
所格は場所や手段を示します。現代日本語の「で」に相当します。また、拡張的に時間を表現するためにも使用されます(例: sorne tæmd: そのときに)。

#### 3.2.5 属格
属格は所有や関係を示します。現代日本語の「の」に相当します。
また、単に限定詞としての機能も持ち、特に指示代名詞の属格は定性の表現にも使用されます(例: sorne kar: その彼)。

#### 3.2.6 出格
出格は起点や比較を示します。現代日本語の「から」に相当します。空間だけでなく、時間的な起点をも示します。

#### 3.2.7 向格
向格は方向や目標を示します。現代日本語の「へ」に相当します。空間だけでなく、時間的な目標や完了時刻をも示します。

#### 3.2.8 共格
現代日本語における随伴動作主を表す「と」や「や」は共格として曲用に吸収されました。
例えば、「私はあなたと行く」は「antet ikew」となります。
動詞の活用は主格によって決定されるため、「antet ikep」や「antet ikel」は「私はあなたと行く」を意味せず、それぞれ「あなたはあなたと行く」や「我々はあなたと行く」を意味します。

## 第6部：動詞の活用体系

### 6.1 動詞の基本構造

ニンゴト語動詞は以下の構造を持ちます：
**法接頭辞 + 接頭辞 + 語幹 + 敬語接辞 + 態標識 + 相標識 + 時制標識 + 人称標識 + 否定接尾辞**

否定は「-ñ」接尾辞で表現されます。例えば、「(私は)行かない」は「ikæwñ」となります。

### 6.2 接頭辞体系

| 現代日本語表現 | 1000年後の接頭辞 | 機能         | 例                            |
| -------------- | ---------------- | ------------ | ----------------------------- |
| もう〜         | **ma-**          | 完了・既遂   | **ma**-jom-ta (もう読んだ)    |
| まだ〜         | **mr-**          | 継続・未完了 | **mr**-jom-a (まだ読んでいる) |
| 再〜/また〜    | **sæ-**          | 再度・反復   | **sæ**-kak-ta (再び書いた)    |

## 第7部：借用語の処理メカニズム

### 7.1 借用語の音韻適応規則

| 適応タイプ         | ルール                               | 例                                          |
| ------------------ | ------------------------------------ | ------------------------------------------- |
| 子音クラスタの分解 | CC → CəC (前舌環境) / CaC (後舌環境) | "system" → "ʃsətem" (前舌) / "sasta" (後舌) |
| 末尾子音の処理     | C# → C + 調和母音                    | "net" → "net-i" (前舌) / "net-a" (後舌)     |
| 非許容音素の代替   | θ → s, ð → z, v → v/b                | "theory" → "sior-i"                         |

### 7.2 借用語の母音調和適応

- **前舌優勢規則**: 漢語・欧米語の借用語は前舌母音系になる傾向が強い
  - 例: "computer" → "kompit" (前舌系)
  - 例: "政治" → "sæʒ" (前舌系)

- **後舌転換規則**: 後舌母音(/a/, /o/, /u/)が多い借用語は後舌系に適応
  - 例: "global" → "glɔbal" (後舌系)
  - 例: "音楽" → "onɔk" (後舌系)

### 7.3 借用語の文法適応

- **名詞借用語**: 語末の音韻によって前舌/後舌のいずれかの曲用パターンに編入
  - 前舌終わり: 前舌名詞曲用 (例: "kompit-i" コンピュータが)
  - 後舌終わり: 後舌名詞曲用 (例: "lɔg-a" ログが)

- **動詞借用語**: "-r/l" 接尾辞を付加して動詞化
  - 例: "click" → "klik-r-" (クリックする)
  - 例: "control" → "kont-rl-" (コントロールする)

### 7.4 複合語内の借用語処理

#### 7.4.1 複合語内の母音調和境界

複合語形成における母音調和の適用規則：

1. **主要部優先規則**（基本）：
   - 後部要素（主要部）の母音タイプが複合語全体を支配
   - 例：çit (前舌) + koto (後舌) → çitakoto (人言葉) 「後舌化」

2. **借用語境界規則**（例外）：
   - 新しい借用語との境界では母音調和が一時中断し再開
   - 例：bunka (後舌) + filosfi (前舌借用語) → bunkafilosfi (文化哲学) 「調和中断」

3. **歴史的複合語規則**（特殊）：
   - 長く使われてきた複合語では全体が調和する傾向
   - 例：gakr (後舌) + çim (前舌) → gakraçom (学問) 「完全調和」

#### 7.4.2 借用層による適応差

| 借用層     | 適応特性           | 例                                    |
| ---------- | ------------------ | ------------------------------------- |
| 古層漢語   | 完全な母音調和適応 | 「学校」→「gakr」(後舌調和)           |
| 中層外来語 | 部分的母音調和     | 「コンピュータ」→「kompit」(前舌優勢) |
| 新層専門語 | 音韻構造の部分保持 | 「algorithm」→「algritm」(混合調和)   |
| 最新層用語 | 最小限の適応       | 「quantum」→「kwantm」(ほぼ原音)      |

#### 7.4.3 借用語接辞化

一部の頻度の高い借用語は接辞化し、固有語語幹と結合します：

| 接辞           | 由来       | 機能       | 例                          |
| -------------- | ---------- | ---------- | --------------------------- |
| **spr-/spir-** | super (超) | 超越・卓越 | **spir**-çit (超人)         |
| **an-/en-**    | anti (反)  | 対立・反対 | **an**-bunka (反文化)       |
| **ml-/mil-**   | multi (多) | 多様・複数 | **mil**-kotba (多言語)      |
| **-loʒ/-leʒ**  | -logy (学) | 学問分野   | kotba**-leʒ** (言語学)      |
| **-tik/-tak**  | -tic (的)  | 性質・特性 | sistem**-tik** (システム的) |

## 第8部：複合語形成規則

### 8.1 主要な複合語形成パターン

1. **直接結合型**: N₁ + N₂ → N₁N₂
   - 子音終わり + 子音始まり → 連結母音挿入
   - 例: çit(人) + tʃuʃn(中心) → çit-i-tʃuʃn (人中心)

2. **属格型**: N₁ + no + N₂ → N₁nN₂
   - 例: jam(山) + no + mç(道) → jamn-mç (山の道)

3. **動詞語幹型**: V-幹 + N → VN
   - 例: jom(読む) + hñ(本) → jom-hñ (読書)

### 8.2 複合時の母音調和規則

- **主要部優先規則**: 後部要素(主要部)の母音タイプが複合語全体の母音調和を決定
  - 例: çit (前舌) + koto (後舌) → çit-a-koto (人言葉) 「後舌化」
  - 例: tæm (後舌) + çim (前舌) → tæm-i-çim (瞬間) 「前舌化」

- **境界弱化**: 複合語境界で母音調和制約が弱まる現象
  - 例: jam-i-mç (山道) 「混合型許容」

### 8.3 形態音韻変化

1. **子音同化**:
   - 例: çit + koto → çit-koto → çik-koto (人言葉)

2. **母音融合**:
   - 例: ki (木) + eda (枝) → ki-eda → kæda (木の枝)

| 標識            | 由来         | 機能                 | 例                                                                |
| --------------- | ------------ | -------------------- | ----------------------------------------------------------------- |
| **-arat/-aret** | 「改まって」 | 公式文体への切り替え | vtʃi **-arat** moʃ-ta-l (私、改まってお願いしました)              |
| **-kdak/-kdek** | 「くだけて」 | 口語文体への切り替え | **-kdak** jom-a ne (くだけて言うと読むね)                         |
| **snoma/sneme** | 「そのまま」 | 引用文体保持         | kar **snoma** "joks-a" to ier-ta (彼はそのまま「いいね」と言った) |

### 12.3 メタ言語標識と引用構造

#### 12.3.1 引用と言及の標識

メタ言語機能を担う標識は、引用助詞や動詞から発達しました：

| 標識          | 由来       | 機能     | 例                                              |
| ------------- | ---------- | -------- | ----------------------------------------------- |
| **to/te**     | 「と」     | 直接引用 | "joks ara" **to** ier-ta (「いいです」と言った) |
| **-tju/-tjo** | 「という」 | 概念言及 | "nihoŋgo" **-tju** kotba (「日本語」という言葉) |
| **-tva/-tve** | 「とは」   | 定義     | "çit" **-tve** nan-zo (「人」とは何か)          |
| **-nad/-ned** | 「など」   | 例示     | çit, hñ **-nad** (人、本など)                   |

#### 12.3.2 引用と思考の融合形式

思考や認識を表す標識：

| 標識          | 由来         | 機能     | 例                                       |
| ------------- | ------------ | -------- | ---------------------------------------- |
| **-tmo/-tme** | 「と思う」   | 思考引用 | joks-a **-tmo** omot-a (いいと思った)    |
| **-tka/-tke** | 「と考える」 | 深層思考 | muzks **-tka** kanre-ta (難しいと考えた) |
| **-tsi/-tsa** | 「とし」     | 仮定     | joks **-tsa** sur-ta (良いとした)        |
| **-tmi/-tma** | 「と見る」   | 見解     | atar **-tmi** mir-ta (新しいと見た)      |

#### 12.3.3 メタ言語引用の文法枠組み

メタ言語機能は以下の構造的枠組みを持ちます：

1. **直接引用構造**：
   - 引用内容 + **to/te** + 伝達動詞
   - "vtʃi ik-a-w" **to** ier-ta (「私は行く」と言った)

2. **間接引用構造**：
   - 従属節 + **-tomo/-tema** + 伝達動詞
   - kar ik-ro **-tomo** ier-ta (彼が行くと言った)

3. **概念言及構造**：
   - 概念 + **-tju/-tjo** + 名詞句
   - "rekiʃ" **tju** gakmon (「歴史」という学問)

## 第15部：統語構造と構文

### 15.1 基本語順

ニンゴト語も現代日本語と同様に主要部後置型のSOV言語です：

**主語＋目的語＋動詞**

- vtʃi hño jomaw (私は本を読む)
- kar çito mirta (彼は人を見た)

### 15.2 関係節構造

関係節は動詞に「-de/-do」という関係節標識を付加して形成します。前舌動詞か後舌動詞かによって異なる形を取ります。

#### 15.2.1 関係節の形成パターン

1. 関係節内の動詞に「-de/-do」を付加
2. 関係節内で修飾される名詞は欠落（ギャップ戦略）
3. 関係節＋被修飾名詞の順に配置

#### 15.2.2 関係節の例

- vtʃi jom-ta**-do** hñ-a (私が読んだ本は)
- jam-no çhoi-kr̃ mir-e**-de** kesi-ka (山の頂上から見える景色は)
- kino vtʃi tosñ-d æta**-do** çit-a (昨日私が図書館で会った人は)

### 15.3 疑問文の形成

#### 15.3.1 疑問詞疑問文

疑問詞を使った疑問文は、疑問詞を通常の位置に置き、文末に疑問マーカー「ka」を付加します：

- tr ant-ñ hñ-o kurre-ta ka? (誰があなたに本をくれましたか？)
- antp dok ik-o-p ka? (あなたはどこへ行きますか？)

#### 15.3.2 極性疑問文（Yes/No疑問文）

Yes/No疑問文も文末に「ka」を付加します：

- antp amr kur-o-p ka? (あなたは明日来ますか？)
- kors hñ-a ant-ñ-o ka? (この本はあなたのですか？)

### 15.4 接続詞と複文構造

#### 15.4.1 主要な接続詞

| 接続詞 | 意味           | 例                                                                      |
| ------ | -------------- | ----------------------------------------------------------------------- |
| dak    | 〜だから、理由 | ama fal-ta **dak**, vtʃi ik-a-ra-w-ñ (雨が降ったから、私は行かなかった) |
| kas    | しかし、逆接   | ama fal-ra **kas**, vtʃi ik-a-w (雨が降っているが、私は行く)            |
| nod    | 〜ので、原因   | hñ-a omosruk-ta **nod**, jom-ta-l (本が面白かったので、読んだ)          |
| soʃ    | そして、並列   | vtʃi hñ-o jom-a-w **soʃ** kar-e kak-a-w (私は本を読む、そして彼は書く)  |

## 第16部：実践会話と例文

### 16.1 基本会話

#### 16.1.1 挨拶と自己紹介

**A**: dobre! antp gzkis arap ka?  
(こんにちは！あなたは元気ですか？)

**B**: dobre! vtʃi joks araw. antpo?  
(こんにちは！私は良い〘元気〙です。あなたは？)

**A**: vtʃo gzkis araw. antne namas nanra ka?  
(私も元気です。あなたの名前は何ですか？)

**B**: vtʃne namas Taro ara. antne namas nanra?  
(私の名前はタロウです。あなたの名前は？)

**A**: vtʃne namas Hanako ara. kors ʃehrñ atra çiti arap ka?  
(私の名前はハナコです。〘あなたは〙この町に新しい人〘この街に新しく来た人〙ですか？)

**B**: hai, vtʃi sanju kino kors ʃehrñ kurataw.  
(はい、私は三週間前にこの町に来ました。)

#### 16.1.2 日常会話

**A**: antp doko ikrap ka?  
(あなたはどこへ行きますか？)

**B**: vtʃi tosñhe ikraw. antpo?  
(私は図書館へ行きます。あなたは？)

**A**: vtʃi kæfeñ ikraw. antp hñro jomrop ka?  
(私はカフェへ行きます。あなたは本を読みますか？)

**B**: hai, vtʃi niŋoto benkjotap hñro jomrop.  
(はい、私は日本語を勉強するために本を読みます。)
