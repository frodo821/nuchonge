# 語彙学的原則

## 定義作成ガイドライン

### 定義の構造
1. **核心的意味**：最も基本的で一般的な意味を最初に記載
2. **拡張的意味**：基本的な意味から派生した二次的な意味を順次記載
3. **特殊用法**：特定の文脈や領域でのみ使用される意味を最後に記載
4. **比喩的用法**：比喩として使用される場合の意味を必要に応じて記載

### 定義の表現スタイル
- 簡潔かつ明確な表現を使用
- 循環的な定義を避ける（定義対象の語を定義の中で使用しない）
- 定義は現代日本語で記述し、必要に応じて例文で用法を補足
- 文法的・意味的制約がある場合は明示的に記載

### 多義語の扱い
- 意味の論理的な発展順序で構成
- 各意味は明確に番号付けし、関連する意味はサブカテゴリーとして構成
- 意味間の関連性や歴史的な発展関係を示す

## 引用と用例の基準

### 例文の選定基準
1. **典型性**：その語の最も典型的な用法を示す例
2. **多様性**：異なる文法的環境や意味での使用を示す例
3. **自然さ**：自然な言語使用を反映した例
4. **文法的完全性**：完全な文または句の形で提示

### 例文の作成方針
- 辞書に既に登録されている語彙を優先的に使用
- 基本的な日常会話や一般的な状況を反映
- 複雑度のレベルに応じた例文の難易度調整
- 複数の文法構造や曲用/活用形を示す例の包含

### 出典情報
- オリジナル例文の場合は「作例」と明記
- 他の例文から借用した場合は出典を明記

## 同義語と反義語の扱い

### 同義語の記述
- 完全な同義語と部分的な同義語を区別
- 語域や文体の違いを明記
- 使用頻度や一般性の違いを示す（一般的/専門的/古語的など）

### 反義語の記述
- 直接的反義語と間接的反義語を区別
- 対立の種類を明記（相補的/段階的/方向的など）
- 対応する意味番号への参照

### 関連語の記述
- 語彙的関連（同じ意味場に属する語）
- 形態的関連（同じ語根から派生した語）
- 文化的関連（同じ文化的文脈で使用される語）

## 異形と変異形の扱い

### 音韻的変異形
- 母音調和による変異形（前舌/後舌環境）
- 音韻環境による同化現象
- 方言的変異や社会的変異

### 文法的変異形
- 不規則な曲用/活用パターン
- 文体による変異（公式/標準/口語）
- 特定の構文環境での形態変化

### 表記上の変異
- 標準的表記と代替表記
- 歴史的な表記の変化
- 新たな借用語の適応過程での表記揺れ

## 語源情報の文書化基準

### 記述の基本構造
- 現代日本語形式の明示
- 音韻変化過程の段階的説明
- 音韻規則の適用例示

### 借用語の語源情報
- 借用元言語の明示
- 借用時期の推定（可能な場合）
- 音韻適応プロセスの説明
- 意味変化の記述（ある場合）

### 造語・複合語の語源情報
- 構成要素の明示
- 構成プロセスの説明
- 意味的動機付けの記述

## 文法情報の記述

### 名詞の文法情報
- 音韻型（前舌/後舌）
- 名詞分類（有生/無生）
- 不規則曲用パターンの有無
- 数や定性に関する特殊規則

### 動詞の文法情報
- 動詞分類（前舌/後舌）
- 活用パターン（規則/不規則）
- 必須・任意の補語構造
- アスペクト・時制制約
- TAMマトリクス（時制・相・法マトリクス）による体系的な活用表示
  - 直説法の時制（現在・過去・未来）と人称（1・2・3人称、単数・複数）の組み合わせ
  - 条件法・命令法などの法形式と人称の組み合わせ
  - 完遂相・結果状態相などのアスペクト形式
  - 代表的な時制-相-法の複合形式

### 形容詞の文法情報
- 形容詞分類（前舌/後舌）
- 名詞修飾パターン
- 比較級・最上級の形成方法
- 副詞的用法の可能性

### その他の品詞
- 特殊な文法的振る舞い
- 文中での位置制約
- 結合可能性の制約

## 用法ラベルシステム

### 時間的ラベル
- 新語（最近導入された語彙）
- 古語（使用頻度が減少している語彙）
- 廃語（現代ではほとんど使用されない語彙）

### 地域的ラベル
- 地域変種の指定
- 方言的特徴

### 社会的・文体的ラベル
- 公式（公式文体で使用）
- 標準（標準文体で一般的）
- 口語（口語文体に特有）
- 俗語（非公式または低い社会的評価）
- 専門（特定の専門分野で使用）

### 頻度ラベル
- 高頻度（日常的に使用される）
- 中頻度（一般的だが日常的ではない）
- 低頻度（まれに使用される）
