## 未解決の課題と優先事項

### レビュー項目
1. **母音調和規則の例外** - 複合語における母音調和の適用方法の明確化が必要
2. **形容詞修飾の一致規則** - 形容詞が異なる音韻型の名詞を修飾する際の規則整理
3. **音節主音の文法化** - 音節主音（/ñ/, /r̃/, /l̃/, /m̃/）の文法機能の一貫性確保
4. **宗教・哲学的概念の語彙体系** - 創世記翻訳で追加した宗教的語彙の体系的整理

### 優先作業項目
1. **基本動詞エントリーの追加** - 次の5語（wrui, hanas, kik, mæ, uʃr）を優先的に追加
2. **例文コレクションの拡充** - より多様なジャンルの複雑な例文の作成
3. **文法機能語の拡充** - 基本的な接続詞、助詞、小辞など文法機能を持つ語彙の追加
4. **語彙セマンティックネットワークの構築** - 関連語のネットワーク整備（特に宗教・哲学的概念の語彙間の関係）

### 長期的な課題
1. **借用語の処理メカニズムの詳細化** - 外来語の音韻適応規則の具体化
2. **文体レベルの詳細な例示** - 公式/標準/口語の各文体における表現の差異の体系化
3. **専門分野語彙の拡充計画** - 科学、芸術、技術などの専門領域の語彙開発戦略
