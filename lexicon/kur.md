# 見出し語: kur

## 文法情報
- **類別**: 動詞
- **音韻型**: 後舌
- **動詞分類**: 自動詞

## 語源情報
現代日本語「くる」(来る)から変化
* /kuru/ → /kur/ と音韻変化
* 語末母音の脱落により語幹が強化された

## 定義
1. ある場所から話者のいる場所や参照地点に移動する
2. 到着する、たどり着く
3. （時間や出来事が）発生する、やってくる
4. （物事が）由来する、起源を持つ

## 例文
1. kar amr æñ kuraw. (彼は明日家に来ます。)
2. antp dokr̃ kurap? (あなたはどこから来ますか？)
3. furaa çiti ʃehrd kurta. (古い人が町に来た。)
4. kors iden vtʃkr̃ kura. (この考えは私から来ている。)

## 関連語
- kuramr: 来訪、訪問（「来る」+「時」の複合語）
- kuræ: 来場所、集合場所（「来る」+「場所」の複合語）
- kurkr̃: 出身、来所（「来る」+「から」の複合語）
- kurçit: 来訪者、訪問者（「来る」+「人」の複合語）
- ik: 行く（反義関連語）

## 備考
「kur」は移動を表す最も基本的な動詞の一つで、「ik」（行く）の反義的な機能を持ちます。話者の視点を基準として、話者の方向への移動を表します。

この動詞は空間的な移動だけでなく、時間的な出来事や抽象的な概念の出現・発生も表現できます。例えば「季節が来る」「思いが来る」などの表現が可能です。

不規則な縮約形も日常会話でよく使われます：
- kraw = kuraw（来る、1人称現在）
- krap = kurap（来る、2人称現在）

特に将来の出来事を予測する場合に頻繁に使用され、「kurhza」（来るはずだ）、「kurkma」（来るかもしれない）などのモダリティ表現との結合も一般的です。
