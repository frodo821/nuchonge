# 見出し語: mono

## 文法情報
- **類別**: 名詞
- **音韻型**: 後舌
- **名詞分類**: 無生

## 語源情報
現代日本語「もの」から直接変化。二音節語でありながら音韻変化による大きな変形を受けずに保存された単語の一つ。

## 定義
1. 物体、実体を持つ存在物
2. 抽象的な事柄、事象
3. （所有関係で）所有物、持ち物
4. （集合的に）全体としての物事
