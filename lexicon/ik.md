# 見出し語: ik

## 文法情報
- **類別**: 動詞
- **音韻型**: 前舌
- **動詞分類**: 前舌動詞

## 語源情報
現代日本語「いく」(行く)から変化
* /iku/ → /ik/ と音韻変化

## 定義
1. ある場所から別の場所へ移動する
2. 時間の経過や事態の推移を表す
3. 物事が進行する、進む
4. （慣用的に）機能する、作動する

## 例文
1. vtʃi æn̩ ikew. (私は家に行きます。)
2. ɔki çitis vtʃne çisi æe miren dak, vtʃi karlne ʃehrs ikrotaw. (大きな人たちが私の小さな家を見ているので、私は彼らの街に行くつもりでした。)
3. antp dokn̩ ikop ka? (あなたはどこに行きますか？)
4. çitri tosn̩he iktan. (人々は図書館へ行きました。)

## 関連語
- kur: 来る（対義語）
- ikkaw: 行く（「ikaw」の縮約形）
- ikrek: 行ける（「ikre ik」の縮約形）
- megik: 巡る（接頭辞meg-「周り」が付加）

## 備考
「ik」は移動を表す最も基本的な動詞の一つです。物理的な移動だけでなく、時間の経過や状態の変化などの抽象的な「移動」も表現します。方向を示す格（向格、与格）と組み合わせて使用されることが多く、目的地や移動の方向を明示します。前舌動詞として規則的な活用パターンに従いますが、縮約形「kaw」（行く）のような不規則形も日常会話では頻繁に使用されます。また「ikm̩ta」（行けば）のように条件形で使われることも多い動詞です。
