# 見出し語: ik

## 文法情報
- **類別**: 動詞
- **音韻型**: 前舌子音
- **動詞分類**: 自動詞
- **活用**: 不規則変化

## 語源情報
現代日本語「いく」(行く)から変化

- /iku/ → /ik/ と音韻変化

## 定義
1. ある場所から別の場所へ移動する
2. 時間の経過や事態の推移を表す
3. 物事が進行する、進む
4. （慣用的に）機能する、作動する
