# 見出し語: joks

## 文法情報
- **類別**: 形容詞
- **音韻型**: 後舌
- **形容詞分類**: 後舌形容詞

## 語源情報
現代日本語「よい」(良い)から変化
* /joi/ → /joi/ → /jok/ → /joks/ と音韻変化
* 語末に「s」が付加される強化現象

## 定義
1. 要求・期待・標準を満たしている、満足できる品質を持つ
2. 道徳的に正しい、倫理的な
3. 喜ばしい、楽しい、幸せな
4. 健康な、具合が良い
5. 熟練した、上手な、有能な
6. 適切な、適当な、ふさわしい

### 前舌名詞修飾時の非対称的曲用
前舌名詞を修飾する場合、形容詞は母音調和の規則に従って「joksi」のように前舌曲用形を取ることもあります。

## 例文
1. joksa çiti kino kurta. (良い人が昨日来ました。)
2. kora hn̩a joksas ara. (この本は良いです。)
3. antp jokso sano tbentap ka? (あなたは良い食事を食べましたか？)
4. vtʃi joksd æd tbrtrtrawa. (私は良い家に住んでいます。)

## 関連語
- joksls: 良さ、善（「良い」+抽象化接尾辞「ls」）
- tvjoks: とても良い（強意接頭辞「tv」+「良い」）
- joksçit: 良い人、善人（「良い」+「人」の複合語）
- jokstben: 美味しく食べる（「良い」+「食べる」の複合語）
- wrui: 悪い（反義語）

## 備考
「joks」は基本的な評価を表す形容詞で、物理的、道徳的、審美的など、様々な文脈で「良い」という評価を表現できます。あらゆる種類の対象（人、物、状況、行動など）に対して使用でき、最も頻繁に使用される形容詞の一つです。

慣用表現として「joks araw」（よろしい、いいです）が肯定的応答として使用されるほか、「joksn̩ aret」（もしよろしければ）のような丁寧な依頼表現にも使われます。

「joks」と「wrui」（悪い）は基本的な評価の対立軸を形成し、多くの文脈で対比的に使用されます。また、強意接頭辞「tv-」と結合して「tvjoks」（非常に良い、素晴らしい）という表現がよく使われます。

感情や状態を示す表現では「vtʃi joksa」（私は元気です、私は調子がいいです）のように使われ、健康状態や気分を尋ねる「antp joksap ka?」（あなたは元気ですか？）は一般的な挨拶表現です。
