# 見出し語: furæ

## 文法情報
- **類別**: 形容詞
- **音韻型**: 後舌
- **活用**: 規則変化

## 語源情報
現代日本語「ふるい」(古い)から変化

- /furui/ → /furu/ → /fur/ → /furæ/ と音韻変化

- 語末の「i」が「æ」に変化する前舌化現象

## 定義
1. 長い時間存在している、製造されてから時間が経っている
2. 最近のものではない、過去の時代に属する
3. 使い古された、摩耗した
4. （習慣・考え方などが）時代遅れの、時代に合わない
