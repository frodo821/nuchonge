# 見出し語: atsmar

## 文法情報
- **類別**: 動詞
- **音韻型**: 後舌
- **動詞分類**: 後舌動詞

## 語源情報
現代日本語の「集まる」(あつまる)から。「あつ」の子音が保持され、母音は後舌化し、「まる」は「mar」に縮約された。

## 定義
1. 散らばっていたものが一箇所に集合する
2. 複数の人や物が同じ場所に集合する
3. 資源や情報などが蓄積される

## 例文
1. sorano ʃta mz̩a atsmar-te umn̩ nar-e (空の下の水は集まって海になれ)
2. çit-ri tosn̩-d atsmar-ta-n (人々は図書館に集まった)
3. vtʃislno æ-d atsmar-ro-l (私たちの家に集まりましょう)
4. mz̩a kubɔm-d atsmar-a (水はくぼみに集まる)
