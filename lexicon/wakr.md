# 見出し語: wakr

## 文法情報
- **類別**: 動詞
- **音韻型**: 後舌子音
- **動詞分類**: 自動詞
- **活用**: 規則変化

## 語源情報
現代日本語の「分ける(わける)」から変化。「わける」→「waker」→「wakr」と音韻変化した。語中母音の「e」が脱落し、子音クラスタが形成された。

## 定義
1. 区別する、分ける（異なるものに区分する）
2. 分離する、切り離す（物理的に離す）
3. 区分する、区画する（空間や時間を分ける）
4. 配分する、割り当てる（何かを分配する）
