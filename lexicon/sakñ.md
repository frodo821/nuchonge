# 見出し語: sakñ

## 文法情報
- **類別**: 名詞
- **音韻型**: 後舌
- **名詞分類**: 無生

## 語源情報
現代日本語の「魚(さかな)」から音韻変化。「さかな」→「sakana」→「sakñ」と変化し、語末母音の脱落と鼻音化により、末尾の「na」が音節主音「ñ」となった。

## 定義
1. 魚（水中に生息する脊椎動物）
2. 魚類（分類学上の魚のグループ）
3. 魚肉、魚料理（食物としての魚）
4. 水中生物（比喩的に水中で生きる生物全般）
