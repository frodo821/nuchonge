# 見出し語: sakñ

## 文法情報
- **類別**: 名詞
- **音韻型**: 後舌
- **名詞分類**: 有生

## 語源情報
現代日本語の「魚(さかな)」から音韻変化。「さかな」→「sakana」→「sakñ」と変化し、語末母音の脱落と鼻音化により、末尾の「na」が音節主音「ñ」となった。

## 定義
1. 魚（水中に生息する脊椎動物）
2. 魚類（分類学上の魚のグループ）
3. 魚肉、魚料理（食物としての魚）
4. 水中生物（比喩的に水中で生きる生物全般）

## 例文
1. kama umñ ari ɔki ikm̃no haʒm, arjr spno sakñ to tɔro tsukrta. (神は海に住む大きな生き物をはじめ、あらゆる種類の魚と鳥を造りました。)
2. kama ierta: "uma sakño kar hkno ikm̃no afrte, soraa arjr spno tɔrñ mitʃe." (神は言われました。「海は魚やその他の生き物であふれ、空はあらゆる種類の鳥で満ちよ。」)
3. sakña umd ojota. (魚は海を泳いだ。)
4. vtʃi sakño tbenaw. (私は魚を食べます。)

## 備考
「sakñ」は創造物語において、五日目に創造された海の生物として重要な位置を占める。特に「arjr spno sakñ」（あらゆる種類の魚）という表現で言及され、生物の多様性を示す代表的な生物群として描かれる。

神の創造物として祝福され、「umippæñ mitʃe」（海いっぱいに満ちよ）という命令を受ける。「sakñ」は基本的な食物源としても言及され、生態系の基盤としての重要性も示唆されている。
