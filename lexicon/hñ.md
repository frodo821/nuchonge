# 見出し語: hñ

## 文法情報
- **類別**: 名詞
- **音韻型**: 後舌
- **名詞分類**: 無生

## 語源情報
現代日本語「ほん」(本)から変化

- /hon/ → /hñ/ と音韻変化

- 母音の脱落と子音の音節主音化が生じた

## 定義
1. 印刷または電子化された文字・画像からなる情報媒体
2. 書籍、図書、出版物
3. 知識や情報の源泉（比喩的用法）
4. 基本、根本、原理（複合語の中で）
