# 見出し語: çirbm

## 文法情報
- **類別**: 動詞
- **音韻型**: 前舌
- **動詞分類**: 配置動詞

## 語源情報
現代日本語の「散りばめる(ちりばめる)」から変化。「ちりばめる」→「chiribameru」→「çirbm」と音韻変化した。「ch」が「ç」に変化し、複数の母音が脱落して子音クラスタが形成された。

## 定義
1. ちりばめる、散りばめる（多数の物を広い範囲に配置する）
2. 点在させる、散在させる（規則性なく配置する）
3. 装飾する、飾る（美的効果を目的として配置する）
4. 広げる、展開する（空間的に分布させる）

## 例文
1. kama karo sorañ çirbmte, tr̃o tersijoñ ʃta. (神はそれを空にちりばめ、地を照らすようにしました。)
2. taijoa hoʃikr̃ soran çirbmtrta. (太陽は星々の間に空にちりばめられた。)
3. taijoçkari tr̃he çirbmta. (太陽の光が地にちりばめられた。)
4. vtʃi hoʃiro tenñ çirbmow. (私は星を天にちりばめるでしょう。)

## 関連語
- hoʃi (星) - 名詞
- taijo (太陽) - 名詞
- tska (月) - 名詞
- sora (空) - 名詞
- kagajk (輝く) - 動詞
- ters (照らす) - 動詞

## 備考
「çirbm」は創造物語において、四日目の創造で神が天体を空間に配置する際に使用される重要な動詞である。特に「kama karo sorañ çirbmte」（神はそれを空にちりばめ）という表現で、星々などの天体が計画的かつ美的に配置されたことを示す。

この動詞は単なる物理的な配置だけでなく、秩序と美を兼ね備えた創造行為としての意味合いを持ち、神の創造の意図的な側面を強調する表現として使用される。多くの小さなものを広い空間に美しく配置するという、芸術的な創造のニュアンスも含んでいる。
