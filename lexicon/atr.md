# 見出し語: atr

## 文法情報
- **類別**: 形容詞
- **音韻型**: 前舌子音
- **活用**: 規則変化

## 語源情報
現代日本語「あたらしい」(新しい)から変化

- /ataraʃii/ → /ataraʃi/ → /atraʃ/ → /atr/ と音韻変化

- 子音クラスタの形成と語末母音の脱落による圧縮

## 定義
1. 最近作られた、または出現した
2. 今までになかった、斬新な性質を持つ
3. 使用されていない、まだ摩耗していない
4. （関係・経験について）最近始まったばかりの
