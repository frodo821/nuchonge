# 見出し語: taijo

## 文法情報
- **類別**: 名詞
- **音韻型**: 前舌子音
- **名詞分類**: 無生

## 語源情報
現代日本語の「太陽(たいよう)」から変化。「たいよう」→「taiyō」→「taijo」と音韻変化した。「y」と「ō」が合わさって「jo」となった。

## 定義
1. 太陽（天体としての太陽）
2. 太陽光、日光（太陽からの光）
3. 中心、中核（比喩的な意味での中心的存在）
4. 光源、輝き（光と熱を発するもの）
