# 見出し語: -dak/-dek

## 文法情報
- **類別**: 接尾辞
- **音韻型**: 母音調和による異形態
- **結合特性**: 名詞・動詞・形容詞に付加可能

## 語源情報
現代日本語の「だけ」（限度や制限を表す助詞）から派生。
* 「だけ」→「dake」→音韻変化と母音調和により「-dak」（後舌語彙用）と「-dek」（前舌語彙用）の二形態が発達

## 定義
1. ～だけ、～のみ、～限り（限定・制限）
2. ～まで、～に至るまで（範囲や限界の指定）
3. ～ほど（程度の限定）

## 用法パターン

### 名詞との結合
- 後舌名詞 + **-dak**: 「san**-dak**」（ご飯だけ）
- 前舌名詞 + **-dek**: 「çit**-dek**」（人だけ）

### 形容詞との結合
- 後舌形容詞 + **-dak**: 「ɔk**-dak**」（大きいだけ）
- 前舌形容詞 + **-dek**: 「çis**-dek**」（小さいだけ）

### 動詞との結合
- 後舌動詞 + **-dak**: 「jom**-dak**」（読むだけ）
- 前舌動詞 + **-dek**: 「mir**-dek**」（見るだけ）

### 数量詞との結合
- 「çt**-dak**」（一つだけ）
- 「sn**-dak**」（三つだけ）

## 音韻変化規則
1. 母音調和による選択:
   - 前舌母音を含む語幹には「**-dek**」
   - 後舌母音を含む語幹には「**-dak**」
   - 混合母音を含む語幹では、最後の音節の母音タイプによって選択

2. 子音終わりの語幹との結合:
   - C + **-dak/-dek** → C**-dak/-dek** (変化なし)
   - 例: çit + -dek → çit-dek (人だけ)

3. 母音終わりの語幹との結合:
   - V + **-dak/-dek** → V**-dak/-dek** (母音融合なし)
   - 例: æ + -dek → æ-dek (家だけ)

## 曲用影響
この接尾辞が付加された語は、元の語幹の品詞を保持しつつ、以下の特性を持ちます:

1. **名詞+dak/dek**: 
   - 名詞として機能し、通常の曲用を受ける
   - 例: çit-dek-i (人だけが)、san-dak-o (ご飯だけを)

2. **形容詞+dak/dek**: 
   - 形容詞的機能を保持
   - 例: çis-dek-i çit-i (小さいだけの人)

3. **動詞+dak/dek**: 
   - 動名詞的性質を獲得
   - 例: jom-dak-a joks-a (読むだけで良い)

## 例文
1. vtʃi san**-dak-o** tbentataw. (私はご飯だけを食べました。)
2. kar hn̩**-dak-a** miro. (彼は本だけを見るでしょう。)
3. çisi çit**-dek-i** korn̩ kurtaw. (小さい人だけがここに来ました。)
4. mir**-dek** næn jom**-dak** matantaw. (見るだけでなく読むだけでも待ちました。)
5. arj-çiti mz̩**-dak-o** nom**-dak** tbentaw. (すべての人々は水だけを飲むだけ食べました。)

## 関連語
- **-smad/-smed**: ～さえ、～すら (極限を表す接尾辞)
- **-mde/-mda**: ～まで (範囲の終点を示す接尾辞)
- **næn**: ～でない (否定)
- **arj-**: すべての、あらゆる (接頭辞)

## 備考
「-dak/-dek」は制限や限定を表す基本的な接尾辞として日常会話で頻繁に使用されます。特に否定表現「næn」と組み合わせて「～だけではない」という表現を形成することが多い。

この接尾辞は文法的に制限を表すだけでなく、哲学的なニュアンスを含む場合もあり、「çit-dek」(人間だけ、人間性の本質)のように概念的な議論でも使用されます。また、「-dak/-dek」を重ねて「-dakdak/-dekdek」とすると「～だけだけ」という強調形になり、より強い限定を表します。
