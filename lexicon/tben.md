# 見出し語: tben

## 文法情報
- **類別**: 動詞
- **音韻型**: 前舌子音
- **動詞分類**: 自動詞
- **活用**: 規則変化

## 語源情報
現代日本語「たべる」(食べる)から変化

- /taberu/ → /taber/ → /tber/ → /tben/ と音韻変化

- 語中母音の脱落と語末の「r」が「n」に変化

## 定義
1. 食物を口に入れて咀嚼し、飲み込む
2. 飲食物を摂取して栄養を得る
3. （比喩的に）何かを徐々に消費する、浪費する
4. （感情・状況に）悩まされる、苦しむ
