# 見出し語: arj-

## 文法情報
- **類別**: 接頭辞
- **音韻型**: 前舌
- **結合特性**: 名詞・動詞・形容詞に付加可能

## 語源情報
形容詞「arj」(すべての、あらゆる)から派生した接頭辞。
現代日本語「あらゆる」から変化した「arj」が文法化して接頭辞となった。
* /arayuru/ → /arajuru/ → /arjuru/ → /arjur/ → /arj/ と音韻変化

## 定義
1. 全ての、あらゆる、完全な（omni-、pan-に相当）
2. 普遍的な、包括的な
3. 極限・最大限の（超-、最-に相当）

## 用法パターン

### 名詞との結合
- **arjçit**: 「万人、すべての人」
- **arjkotba**: 「万言、あらゆる言語」
- **arjtæm**: 「全時間、永遠」

### 形容詞との結合
- **arjoks**: 「完全に良い、至良の」
- **arjɔk**: 「最大の、極大の」
- **arjçis**: 「微小の、極小の」

### 動詞との結合
- **arjʃik**: 「全てを知る」

## 音韻変化規則
1. 子音で始まる語と結合する場合:
   - **arj-** + C → **arj-** (接頭辞の形態は変化せず)
   - 例: arj + çit → arjçit (すべての人)

2. 母音で始まる語と結合する場合:
   - **arj-** + V → **arj-** (接頭辞の形態は変化せず)
   - 例: arj + ik → arjik (あらゆる場所へ行く)

3. 母音調和の規則:
   - 前舌母音を含む語幹に付加する場合でも、後舌母音を含む語幹に付加する場合でも形態は変化しない

## 例文
1. **arjçiti** korso kotbao hjæratan. (すべての人々がこの言葉を聞いた。)
2. kama **arjʃiko** çita ara. (神は全知の人である。)
3. **arjkio** mori ætataw. (あらゆる木が森を形成した。)
4. **arjtæmd** korsa kotbais hjæro. (永遠の中でこの言葉は聞こえるだろう。)
5. **arjoksi** sanei tabendek vtʃka hjæmztaw. (完全に良い食事だけが私の心を満たした。)

## 関連語
- **arj**: すべての、あらゆる (形容詞)
- **çuʃts**: いくつかの、幾つかの (部分を示す形容詞)
- **-dak/-dek**: だけ、限界や限り、制限 (接尾辞)
