# 見出し語: ikimono

## 文法情報
- **類別**: 名詞
- **音韻型**: 前舌
- **名詞分類**: 有生

## 語源情報
現代日本語の「生き物(いきもの)」から直接変化。「いきもの」→「ikimono」とほぼそのままの形で変化し、音韻構造が保持された。

## 定義
1. 生き物、生物（生命を持つ存在全般）
2. 動物（特に動く生物）
3. 生態系（生物とその生息環境の総体）
4. 魂、精神（比喩的に生命の本質）

## 例文
1. kama umn̩ ari ɔki ikimonoo haʒm, arjr spno sakana to torio tsukrta. (神は海に住む大きな生き物をはじめ、あらゆる種類の魚と鳥を造りました。)
2. kama mata ierta: "tr̩a, kettæ ja tr̩o hai mono, janono ikimono nad, arjr spno ikimonoo ume." (神はまた言われました。「地は、家畜や地をはうもの、野の獣など、あらゆる種類の生き物を生み出せ。」)
3. ikimonori mn ʃbarʃi mono ɔnl arta. (生き物はみなすばらしいものばかりでした。)
4. vtʃi ikimonoo miraw. (私は生き物を見ます。)

## 関連語

- tori (鳥) - 名詞
- kettæ (家畜) - 名詞
- dɔbutsu (動物) - 名詞
- tsukr (創る) - 動詞
- um (生み出す) - 動詞

## 備考
「ikimono」は創造物語において、神によって創造された生命を持つ存在全般を指す総称として使用される。特に五日目と六日目の創造において、海の生物、空の生物、地上の生物という順序で創造されることが描かれる。

「ikimono」という語彙は「生きる」と「物」という概念の合成であり、生命を持つ物質的存在という意味合いを持つ。植物から動物まで幅広い生物を包括する概念であり、創造物語では特に神による祝福の対象として描かれる。
