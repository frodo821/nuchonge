# 見出し語: hoʃi

## 文法情報
- **類別**: 名詞
- **音韻型**: 後舌
- **名詞分類**: 無生

## 語源情報
現代日本語の「星(ほし)」から変化。「ほし」→「hoshi」→「hoʃi」と音韻変化した。「sh」が「ʃ」に変化したが、その他の音はほぼ保持されている。

## 定義
1. 星、恒星（夜空に輝く天体）
2. 星座（複数の星が形作る模様）
3. 運命、宿命（占星術的な意味での星）
4. きらめき、輝き（星のような光や印）
