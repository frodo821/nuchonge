# 見出し語: fue

## 文法情報
- **類別**: 動詞
- **音韻型**: 前舌
- **動詞分類**: 状態変化動詞

## 語源情報
現代日本語の「増える(ふえる)」から変化。「ふえる」→「fueru」→「fue」と音韻変化した。語末母音が脱落し、語幹の形態になった。

## 定義
1. 増える、増加する（数量が多くなる）
2. 繁殖する、繁栄する（生物が数を増やす）
3. 広がる、拡大する（範囲や規模が大きくなる）
4. 発展する、成長する（規模や質が発達する）

## 例文
1. "umippæñ mitʃe. tɔriraa tr̃o ɔirsmdñ fue" to ivæta. (「海いっぱいに満ちよ。鳥たちは地を覆うまでに増えよ」と祝福しました。)
2. ikm̃n tr̃d fueta. (生き物が地に増えた。)
3. sakñhkno ikm̃n uma mitʃastñ fuea. (魚などの生き物が海に満ちるほど増えている。)
4. ça skmino makistñ çitri fueta. (日ごとの長さとともに人々は増えた。)

## 関連語
- mitʃ (満ちる) - 動詞
- afr (あふれる) - 動詞
- ɔir (覆う) - 動詞
- çit (人) - 名詞
- ikm̃n (生き物) - 名詞
- sakñ (魚) - 名詞
- tɔr (鳥) - 名詞

## 備考
「fue」は創造物語において、特に神の祝福に関連して使用される重要な動詞である。「tɔriraa tr̃o ɔirsmdñ fue」（鳥たちは地を覆うまでに増えよ）のように、創造された生物の繁栄と発展を命じる文脈で使用される。

「fue」は「mitʃ」（満ちる）、「afr」（あふれる）といった豊穣を表す動詞と共に使われることが多く、創造の目的の一つである生命の繁栄を示す重要な概念である。数の増加だけでなく、領域の拡大や存在感の強化といった側面も含む。
