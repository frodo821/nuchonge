# 見出し語: san

## 文法情報
- **類別**: 名詞
- **音韻型**: 後舌
- **名詞分類**: 無生

## 語源情報
現代日本語「ごはん」(ご飯)から変化

- /gohan/ → /gohan/ → /gan/ → /san/ と音韻変化

- 敬称接頭辞「ご」との融合と子音変化

## 定義
1. 炊いた米、米飯
2. 一般的な食事、食べ物
3. 食事の時間、食事の機会
4. （比喩的に）生計、暮らし
