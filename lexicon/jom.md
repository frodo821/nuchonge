# 見出し語: jom

## 文法情報
- **類別**: 動詞
- **音韻型**: 後舌
- **動詞分類**: 後舌動詞

## 語源情報
現代日本語「よむ」(読む)から変化
* /jomu/ → /jom/ と音韻変化

## 定義
1. 文字や記号を見て、その意味を理解する
2. 書物などを読んで理解する
3. 心や状況を推測して理解する
4. データを読み取る、アクセスする

## 例文
1. vtʃi hño jomaw. (私は本を読みます。)
2. vtʃi jomtado hña çitine karlto kuratado kɔrmra. (私が読んだ本は、人々が彼らと一緒に来た物語だ。)
3. antp kors hño jomtap ka? (あなたはこの本を読みましたか？)
4. çiti kotbao joma dak, ʃikoa fkaro. (人は言葉を読むので、思考が深くなるだろう。)

## 関連語
- jomhñ: 読書（「読む」+「本」の複合語）
- jkjom: よく読む（接頭辞jk-「習慣・頻度」が付加）
- sæjom: 再読する（接頭辞sæ-「再度・反復」が付加）
- jomtmi: 読み取る（「読む」+「見なす」の複合語）

## 備考
「jom」は知識獲得や情報処理の基本動詞です。物理的なテキスト読解だけでなく、状況や気持ちを「読む」という抽象的な意味でも使用されます。後舌動詞として規則的な活用パターンに従います。対格で表される直接目的語（何を読むか）を取ることが多く、特に「hñ」（本）との結合頻度が高い動詞です。また、関係節の中でよく使用される動詞でもあり、「jomtado hñ」（読んだ本）のような表現が一般的です。
