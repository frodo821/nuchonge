# 見出し語: hirgr

## 文法情報
- **類別**: 動詞
- **音韻型**: 前舌子音
- **動詞分類**: 自動詞
- **活用**: 規則変化

## 語源情報
現代日本語の「広がる」(ひろがる)から。「ひろ」の子音が保持され、母音は前舌化し、「がる」は「gr」に縮約された。

## 定義
1. 空間的に拡張する、範囲が大きくなる
2. (生物が)数を増やして領域を拡大する
3. 影響や効果が及ぶ範囲が大きくなる
4. 知識や考えが普及する
