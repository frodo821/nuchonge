# 見出し語: hirgr

## 文法情報
- **類別**: 動詞
- **音韻型**: 前舌
- **動詞分類**: 前舌動詞

## 語源情報
現代日本語の「広がる」(ひろがる)から。「ひろ」の子音が保持され、母音は前舌化し、「がる」は「gr」に縮約された。

## 定義
1. 空間的に拡張する、範囲が大きくなる
2. (生物が)数を増やして領域を拡大する
3. 影響や効果が及ぶ範囲が大きくなる
4. 知識や考えが普及する

## 例文
1. tr̃ñ fue hirgr-te (地に増え広がれ)
2. ʃitsr-ri tr̃-d hirgr-ta-n (植物たちは地に広がった)
3. kono koten-a tr̃-ne smkr̃smdñ hirgr-ta (この考え方は世界の隅々まで広がった)
4. m̃za tr̃-ne uæ-d hirgr-e (水が地の上に広がっている)

## 関連語
- **fue**: 増える
- **ɔir**: 覆う
- **atsmar**: 集まる (対義語として)
- **tszk**: 続く

## 備考
創造物語において、hirgr（広がる）は神の祝福と命令の中核をなす概念として登場する。特に、生物が「増えて広がる」ことは、創造の目的の一つとして示されている。空間的な拡大だけでなく、数の増加と共に領域を拡大していくという概念を含んでおり、創造された世界を満たしていくという神の計画を表している。
