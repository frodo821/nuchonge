# 見出し語: afr

## 文法情報
- **類別**: 動詞
- **音韻型**: 後舌
- **動詞分類**: 状態変化動詞

## 語源情報
現代日本語の「溢れる(あふれる)」から変化。「あふれる」→「afureru」→「afr」と音韻変化した。語中および語末母音が脱落し、子音クラスタが形成された。

## 定義
1. あふれる、溢れる（容器や空間から外に出るほど多くなる）
2. 充満する、満ちあふれる（豊かに広がる）
3. 豊富である、豊かである（量や数が多い状態）
4. 横溢する、溢れ出る（感情や印象が強く表れる）

## 例文
1. kama ierta: "uma sakana to kar hkno ikimonon̩ afrte, soraa arjr spno torin̩ mitʃe." (神は言われました。「海は魚やその他の生き物であふれ、空はあらゆる種類の鳥で満ちよ。」)
2. uma ikimono to sakanadn̩ afrta. (海は生き物と魚であふれた。)
3. çkari tenn̩ afrta. (光が天にあふれた。)
4. korske uma afr-sm-ta ikimonod. (こうして海はあふれんばかりの生き物で。)

## 備考
「afr」は創造物語において、特に五日目の創造で生物の豊かさと繁栄を表現する重要な動詞である。命令「uma sakana to kar hkno ikimonon̩ afrte」（海は魚やその他の生き物であふれ）は神の創造的命令の一つとして、生命の繁栄を指示している。

「afr」はしばしば「mitʃ」（満ちる）と対をなし、「afr」がより活動的で勢いのある状態を表すのに対し、「mitʃ」はより静的な充満状態を表す傾向がある。ともに神の創造の豊かさと完全さを表現する重要な語彙である。
