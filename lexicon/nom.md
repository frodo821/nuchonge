# 見出し語: nom

## 文法情報
- **類別**: 動詞
- **音韻型**: 後舌子音
- **動詞分類**: 自動詞
- **活用**: 規則変化

## 語源情報
現代日本語「のむ」(飲む)から変化

- /nomu/ → /nom/ と音韻変化

- 語末母音の脱落による単純化

## 定義
1. 液体を口に入れて飲み込む
2. 薬や栄養剤などを口から摂取する
3. （比喩的に）知識や情報を取り入れる
4. 煙や煙霧などを吸い込む
5. （感情・状況を）受け入れる、我慢する
