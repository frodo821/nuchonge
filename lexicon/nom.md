# 見出し語: nom

## 文法情報
- **類別**: 動詞
- **音韻型**: 後舌
- **動詞分類**: 他動詞

## 語源情報
現代日本語「のむ」(飲む)から変化
* /nomu/ → /nom/ と音韻変化
* 語末母音の脱落による単純化

## 定義
1. 液体を口に入れて飲み込む
2. 薬や栄養剤などを口から摂取する
3. （比喩的に）知識や情報を取り入れる
4. 煙や煙霧などを吸い込む
5. （感情・状況を）受け入れる、我慢する

## 例文
1. vtʃi m̃zo nomaw. (私は水を飲みます。)
2. çiti jaksno nomta. (人は薬を飲んだ。)
3. antp kino sakeo nomtap ka? (あなたは昨日酒を飲みましたか？)
4. kara nomtasma hño jome. (彼は飲み終わってから本を読む。)

## 関連語
- nommn: 飲み物（「飲む」+「もの」の複合語）
- nomdsm: 飲み始める（「飲む」+起動相「dsm」接尾辞）
- nomsm: 飲み切る（「飲む」+完遂「sm」接尾辞）
- tben: 食べる（関連基本動詞）
- tbenom: 食べ飲み（「食べる」+「飲む」の複合語）

## 備考
「nom」は日常的な基本動詞で、飲食に関する表現に必須です。後舌母音系動詞として、活用形に後舌母音が現れます。丁寧表現では「nommasaw」（飲みます）などの形が使われます。「jknom」（よく飲む、大酒飲みである）のように習慣・頻度を表す接頭辞「jk-」と組み合わせることも一般的です。

身体活動や生理現象を表す基本的な動詞として、様々なアスペクト標識と組み合わせて使用されます：
- nomtra (飲んでいる)
- nomsmta (飲んでしまった)
- nomtko (飲んでおこう)

比喩的には「知識を飲む」（知識を吸収する）、「怒りを飲む」（怒りを抑える）などの表現にも使用されます。また、「m̃z」（水）や「sake」（酒）などの液体名詞と共に使われることが多く、これらの名詞と組み合わせて複合語を形成することもあります（「m̃znom」（水飲み）、「sakenom」（酒飲み）など）。
