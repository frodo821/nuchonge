# 見出し語: æ

## 文法情報
- **類別**: 名詞
- **音韻型**: 前舌
- **名詞分類**: 無生

## 語源情報
現代日本語「いえ」(家)から変化
* /ie/ → /je/ → /æ/ と音韻変化
* 二重母音/ie/が単一母音/æ/に融合

## 定義
1. 人が住むための建物、住居
2. 家族や家系を表す社会的単位
3. （比喩的に）慣れ親しんだ場所や環境

## 例文
1. çiti mire vtʃne æe ka. (人は私の家を見ますか。)
2. vtʃi æn̩ ikew. (私は家に行きます。)
3. æi ɔki ara. (家は大きいです。)
4. kara ækr̩ kurta. (彼は家から来ました。)

## 関連語
- æmç: 玄関（「家」+「道」の複合語）
- æçit: 家族（「家」+「人」の複合語）
- ɔkæ: 大邸宅（「大きい」+「家」の複合語）

## 備考
「æ」は住居と家族の両方を意味する基本的な語彙です。前舌母音系の無生名詞として、規則的な曲用パターンに従います。建物としての物理的な家だけでなく、社会的単位としての家族や帰属意識の対象を表す比喩的な意味でも使用されます。「æ」を基にした複合語も多く、住居環境に関する語彙の基礎となっています。
