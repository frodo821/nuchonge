# 見出し語: ikm̃n

## 文法情報
- **類別**: 名詞
- **音韻型**: 前舌子音
- **名詞分類**: 無生

## 語源情報
現代日本語の「生き物(いきもの)」から変化。「いきもの」→「ikimono」→「ikm̃n」と音韻変化し、子音クラスタ化と音節主音の形成によって現在の形になった。特に、「生きる」(ikiru)の語幹部分「ik」と「物」(mono)の頭子音「m」が結合し、音節主音ñが形成された。

## 定義
1. 生き物、生物（生命を持つ存在全般）
2. 動物（特に動く生物）
3. 生態系（生物とその生息環境の総体）
4. 魂、精神（比喩的に生命の本質）
