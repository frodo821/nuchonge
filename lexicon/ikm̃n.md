# 見出し語: ikm̃n

## 文法情報
- **類別**: 名詞
- **音韻型**: 前舌
- **名詞分類**: 有生

## 語源情報
現代日本語の「生き物(いきもの)」から変化。「いきもの」→「ikimono」→「ikm̃n」と音韻変化し、子音クラスタ化と音節主音の形成によって現在の形になった。特に、「生きる」(ikiru)の語幹部分「ik」と「物」(mono)の頭子音「m」が結合し、音節主音ñが形成された。

## 定義
1. 生き物、生物（生命を持つ存在全般）
2. 動物（特に動く生物）
3. 生態系（生物とその生息環境の総体）
4. 魂、精神（比喩的に生命の本質）

## 例文
1. kama umñ ari ɔki ikm̃nre haʒm, arjr spno sakñ to tɔro tsukrta. (神は海に住む大きな生き物をはじめ、あらゆる種類の魚と鳥を造りました。)
2. kama mata ierta: "tr̃a, kettæ ja tr̃o hai mono, janono ikm̃nre nad, arjr spno ikm̃nre ume." (神はまた言われました。「地は、家畜や地をはうもの、野の獣など、あらゆる種類の生き物を生み出せ。」)
4. vtʃi ikm̃nre miraw. (私は生き物を見ます。)

## 関連語

- dɔbt (動物) - 名詞
- tsukr (創る) - 動詞
- um (生み出す) - 動詞

## 備考
「ikm̃n」は創造物語において、神によって創造された生命を持つ存在全般を指す総称として使用される。特に五日目と六日目の創造において、海の生物、空の生物、地上の生物という順序で創造されることが描かれる。

「ikm̃n」という語彙は「生きる」と「物」という概念の合成であり、生命を持つ物質的存在という意味合いを持つ。植物から動物まで幅広い生物を包括する概念であり、創造物語では特に神による祝福の対象として描かれる。
