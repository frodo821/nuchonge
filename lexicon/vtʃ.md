# 見出し語: vtʃ

## 文法情報
- **類別**: 代名詞
- **音韻型**: 前舌
- **代名詞分類**: 1人称単数

## 語源情報
現代日本語「わたし」(私)から変化
* /watashi/ → /watʃi/ → /vtʃi/ → /vtʃ/ と音韻変化

## 定義
1. 話者自身を指す第一人称代名詞

## 曲用表

| 格   | 形態     | 現代日本語対応  |
| ---- | -------- | --------------- |
| 主格 | vtʃi     | 私は/私が       |
| 対格 | vtʃe     | 私を            |
| 与格 | vtʃn̩     | 私に            |
| 属格 | vtʃne    | 私の            |
| 出格 | vtʃer    | 私から          |
| 向格 | vtʃen̩    | 私へ            |
| 共格 | vtʃet    | 私と            |

*注: 人称代名詞は人が場所たりえないため、所格形式を持ちません。

## 複数形
複数形「vtʃisl」（私たち）は別の見出し語として扱われます。

## 例文
1. vtʃi hn̩o jomew. (私は本を読みます。)
2. çiti mire vtʃne æe ka. (人は私の家を見ますか。)
3. kar vtʃen̩ kakta. (彼は私に書いた。)
4. vtʃer nano mirep ka. (私から何を見ますか。)

## 関連語
- vtʃisl: 私たち（1人称複数形）
- vtʃraw: 私です（「vtʃi araw」の縮約形）
- vtʃvese: 私は（対比トピック標識付き）

## 備考
「vtʃ」は最も基本的な第一人称代名詞で、フォーマルな場面からカジュアルな会話まで幅広く使用されます。文化的に日本語の「わたし」からの変化を反映し、個人のアイデンティティを表現する重要な語彙です。人称代名詞特有の曲用パターンを持ち、特に人が場所になり得ないという概念から所格形式を持ちません。
