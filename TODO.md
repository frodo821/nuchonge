# ニンゴト語文法プロファイル — 次のタスク

## 文法設計（未決定事項）

### 動詞活用
- [x] **前舌母音語幹動詞クラス (verb_front_v)**: 完了。語幹末母音が時制で交替（現在 e, 過去 a, 未来 æ）。連結母音不要。fue, ivæ をテスト語彙として実装
- [x] **後舌母音語幹動詞クラス (verb_back_v)**: 完了。語幹末母音が時制で交替（現在 u, 過去 ɔ, 未来 o）。omɔ をテスト語彙として実装
- [x] **中立母音語幹動詞クラス (verb_neutral)**: 完了。交替 ə/a/a、過去は -t 保持で未来と弁別。2sg: 現在 əj（滑音挿入）、過去 a→e（上昇）、未来 -p（-t との衝突回避で調音部位変更、antp の -p と同源）。CC 相接辞は現在 2sg で CV 転倒。æta をテスト語彙として実装
- [x] **コピュラ ar の新人称体系への対応**: 完了。現在/過去: 3sgがir類推でa→e correction、2sgは原形保持。未来: 2sgがə→u置換。3pl -r、否定 -ñ に統一
- [x] **ik の新人称体系への対応**: 完了。verb_front_cons への delegation で新人称体系が自動適用されることを検証済み
- [x] **アスペクト接辞の歴史的根拠**: 完了。sm←しまう, tr←ている, tk←ておく, ds←だす, v←おえる。補助動詞の子音骨格が残存。booklet_4th.md と verb_inflections.hu に記載
- [x] **連結母音 a/ɔ の歴史的根拠**: 完了。元は全て中立 a。後舌環境で母音調和強化により a→ɔ に同化。名詞の挿入母音 i/u は別地層で発達

### 名詞曲用
- [x] **æ (家) の語幹分類**: 完了。規則クラスから外し inline inflect に変更。/ie/→/æ/ 融合前の /i/ が格接辞に再分析（-id, -it, -im 等）、nom/acc 無標合流、pl は古形 æje
- [ ] **ゼロ主格**: 高頻度名詞で裸語幹=主格のパターン。æ で確立済み。他の候補は具体例が出てきたら個別検討
- [ ] **所格・共格の不規則性の具体化**: -de/-do, -te/-to の化石化母音は導入済み。æ は個別処理で解消。他は具体例が出てきたら再検討

### 形容詞
- [ ] **独自の曲用パターン**: 現在は名詞への delegation だが、md のçis/furæ は名詞パターンと不一致。形容詞独自の変化体系を設計
- [x] **比較級・最上級**: 完了。比較級は接辞 -jər（← より /jori/、ə に弱化で調和中立）。比較対象は出格で標示。最上級は sə + 比較級（定冠詞+比較級、ロマンス語型の分析的構成）

### 音韻体系
- [x] **最大音節構造の改訂**: 完了。booklet_4th.md 1.3 を CVC(C) に更新
- [x] **nñ/ñn/m̃m/m̃n の縮約規則**: 完了。booklet_4th.md 1.3 に鼻音連続の縮約規則を追加、CVC(C) 制約の維持メカニズムとして位置づけ
- [x] **母音調和の再検討**: 完了。a は /ä/（中舌広母音）で ə と同じく中立。前舌(i,e,æ)/後舌(u,o,ɔ)/中立(ə,a) の三分。hae 等の調和違反が解消
- [ ] **否定鼻母音化と子音音節核**: 音節核が子音（/r̩/, /l̩/, /m̩/, /n̩/）の場合、現行の nasalization ルールは母音のみ対象で空振りする。シュワー挿入等の処理を検討（母音語幹動詞や縮約形の追加時に必要になる可能性）

### 代名詞・指示詞
- [x] **代名詞の CVC(C) 整合**: 完了。1sg vtʃ→vəç（破擦音弱化）、antp→ant（無標主格）、1pl vtʃisl→vəçl（「わたしら」基底、-ra→-l）。人称標識語源も更新。日本語ら行は障害なき限り /l/ で実現、3sg kar は 3pl との -l 衝突回避で /r/ 保持
- [x] **定冠詞の発達**: 完了。中称指示詞属格 sorne → sorn → sən → sə に縮約。前接語・不変化詞として名詞の前に付く（sə çiti = その人が）
