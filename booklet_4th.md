# ニンゴト語ブックレット(第4版)
## 音韻体系から文法構造まで

---

## 序論

ニンゴト語は一般的な日本語から1000年後に発達した言語とされていますが、本ブックレットではその言語学的特徴を体系的に記述します。本書は、ニンゴト語の文法体系を包括的に説明する目的で編纂されています。特に、データベースに格納された曲用・活用パターンと実際の言語使用の関係に焦点を当てています。

### 1.3 音節構造

基本的な音節の構造は (C)(C)(C)V(C)(C) であり、最大音節構造 CCCVCC までが許容されます。また、/r̃/, /l̃/, /ñ/, /m̃/ の4つの子音は、音節主音としての機能を持ち、母音の代わりに音節核となることができます。

許容される最大の子音クラスタである CCC は、構成する子音のいずれかが聞こえ度の高い子音である場合に限られ、聞こえ度の低い子音のみで構成されたクラスタは許容されません。

### 1.4 母音調和

ニンゴト語には母音調和の原理が存在し、単語内の母音は前舌母音群（i, e, æ）か後舌母音群（a, o, u, ɔ）のいずれかに属します。中舌母音（ə）は中立的で、どちらの群にも共存できます。

母音調和は接辞にも適用され、語幹の母音群に応じて適切な形態が選択されます。

## 第3部：名詞の曲用体系

### 3.1 名詞分類

名詞は以下の基準で分類されます：

1. **母音調和による分類**
   - 前舌母音系名詞（i, e, æを含む語幹）
   - 後舌母音系名詞（o, u, ɔを含む語幹）
   - a語幹名詞（前舌でも後舌でもなく、aを含む語幹）
   - ə語幹名詞（前舌でも後舌でもなく、əを含む語幹）

2. **文法性による分類**
   - 有生名詞
   - 無生名詞

3. **数による分類**
   - 単数
   - 複数

4. **定性による分類**
   - 不定
   - 定

### 3.2 格体系

ニンゴト語は8つの基本格を持ちます：

| 格   | 機能             |
| ---- | ---------------- |
| 主格 | 動作・状態の主体 |
| 対格 | 動作の対象       |
| 与格 | 間接目的語       |
| 所格 | 場所・手段       |
| 属格 | 所有・関係       |
| 出格 | 起点・比較       |
| 向格 | 方向・目標       |
| 共格 | 共同・相互関係   |

### 3.3 曲用パターン

曲用接尾辞の順序は以下の通りです：
**語幹 + 格標識 + 複数標識 + 定標識**

一般的には「格標識 + 複数標識 + 定標識」が融合した曲用語尾が使用されます。

#### 3.3.1 前舌母音系名詞

**有生名詞の曲用パターン**

- 単数/複数区分
- 定/不定区分
- 複数形成時の音韻調整：一部の格では接続母音を挿入

**無生名詞の曲用パターン**

- 基本的に有生名詞と同じ曲用を示すが、複数形での属格形成に差異がある

#### 3.3.2 後舌母音系名詞

**有生名詞の曲用パターン**

- 前舌系と同様の基本構造
- 対応する母音調和形を使用
- 一部の格では独自の形態を持つ

**無生名詞の曲用パターン**

- 複数形での所格、出格、向格、共格に接続母音「-a」を挿入

## 第4部：形容詞の曲用体系

### 4.1 形容詞分類

形容詞は名詞と同様に以下のように分類されます：

1. **母音調和による分類**
   - 前舌母音系形容詞
   - 後舌母音系形容詞

2. **修飾対象による分類**
   - 有生名詞修飾形
   - 無生名詞修飾形

### 4.2 形容詞の一致

形容詞は修飾する名詞と以下の特性において一致します：

- 数（単数/複数）
- 定性（定/不定）
- 格
- 有生性

### 4.3 曲用パターン

形容詞の曲用は基本的に名詞の曲用パターンに従いますが、修飾対象によって異なる形態をとることがあります。

#### 4.3.1 前舌母音系形容詞

**有生名詞修飾形**
- 修飾対象の名詞と完全に一致する曲用を示す

**無生名詞修飾形**
- 複数属格形成において名詞と異なるパターンを示す

#### 4.3.2 後舌母音系形容詞

**有生名詞修飾形**
- 修飾対象の名詞と完全に一致する曲用を示す

**無生名詞修飾形**
- 基本的に有生名詞修飾形と同じだが、接続母音に差異がある場合がある

## 第5部：代名詞体系

### 5.1 人称代名詞

人称代名詞は、一般の名詞とは異なる独自の曲用パターンを持ちます。人称代名詞は所格を欠きます。

基本形式：
- 1人称単数：vtʃi
- 2人称単数：antp
- 3人称単数：kar
- 1人称複数：vtʃisl
- 2人称複数：antes
- 3人称複数：karl

人称代名詞は場所や時間を表すことがないため、所格は存在しません。

### 5.2 指示代名詞

指示代名詞は距離の3段階区分を持ちます：
- 近称：ko- 系統
- 中称：so- 系統
- 遠称：a- 系統

また、疑問・不定形式として do- 系統があります。

| 種類           | 主格  | 対格  | 与格  | 属格   | 所格  | 出格   | 向格  | 共格  |
| -------------- | ----- | ----- | ----- | ------ | ----- | ------ | ----- | ----- |
| 近称単数       | kori  | kore  | korñ  | korne  | kok   | kokr   | kokñ  | kort  |
| 近称複数       | korir | korer | korñr | korner | kokid | kokikr̃ | kokiñ | korit |
| 中称単数       | sori  | sore  | sorñ  | sorne  | sok   | sokr   | sokñ  | sork  |
| 中称複数       | sorir | sorer | sorñr | sorner | sokid | sokikr̃ | sokiñ | sorit |
| 遠称単数       | ari   | are   | arñ   | arne   | akr   | akr    | akñ   | akt   |
| 遠称複数       | arir  | arer  | arñr  | arner  | akrad | akrakr̃ | akñiñ | aktit |
| 疑問・不定単数 | dori  | dore  | dorñ  | dorne  | dok   | dokr   | dokñ  | dort  |
| 疑問・不定複数 | dorir | dorer | dorñr | dorner | dokid | dokikr̃ | dokiñ | dorit |

## 第6部：動詞の活用体系

### 6.1 動詞分類

動詞は以下のように分類されます：

1. **母音調和による分類**
   - 前舌母音系動詞
   - 後舌母音系動詞

2. **活用の規則性による分類**
   - 規則動詞
   - 不規則動詞

### 6.2 動詞の構造

動詞は以下の構造を持ちます：
**接頭辞 + 法接頭辞 + 語幹 + 活用語尾 + 敬語接辞 + 態標識 + 否定接尾辞**

### 6.3 接頭辞体系

様々な文法的・意味的機能を持つ接頭辞があり、動詞語幹の前に付加されます。

### 6.4 活用語尾
活用語尾はTAM (時制・アスペクト・モダリティ)のうち、時制とアスペクトを表現します。
また、主格名詞の数および人称も表現します。

### 6.5 法（モダリティ）接頭辞

| 現代日本語表現     | 接頭辞   | 法         | 例                               |
| ------------------ | -------- | ---------- | -------------------------------- |
| 〜                 | (ゼロ)   | 直説法     | 事実の叙述                       |
| 〜なければならない | **bək-** | 義務法     | **bək**-ik-a (行かねばならない)  |
| 〜かもしれない     | **kəm-** | 可能法     | **kəm**-fal-a (降るかもしれない) |
| 〜はずだ           | **həz-** | 推定法     | **həz**-kur-a (来るはずだ)       |
| 〜たがる           | **tag-** | 希求法     | **tag**-jom-a (読みたがる)       |
| 〜そうだ（様態）   | **səj-** | 様態推量法 | **səj**-ɔk-a (大きそうだ)        |
| 〜そうだ（伝聞）   | **sak-** | 伝聞法     | **sak**-kur-a (来るそうだ)       |
| 〜であれば         | **rə-**  | 条件法     | **rə**-kur-a (来たならば)        |
| 〜しろ             | **ja-**  | 命令法     | **ja**-kur-a (来い)              |

### 6.6 否定

否定は接尾辞「-ñ」で表現されます。

### 6.7 準動詞

ニンゴト語には以下の準動詞形式があります：
- *-kot/-ket: 名詞化（後舌/前舌、母音語末形）
- *-əkot/-əket: 名詞化（後舌/前舌、子音語末形）

準動詞は名詞的用法と形容詞的用法を持ち、それぞれの機能に応じた曲用を示します。

## 第7部：統語構造

### 7.1 基本語順

ニンゴト語は主要部後置型のSOV言語です：
**主語 + 目的語 + 動詞**

### 7.2 関係節構造

関係節は動詞に「-de/-do」という関係節標識を付加して形成します：
**関係節 + 被修飾名詞**

関係節内で修飾される名詞は欠落（ギャップ戦略）します。

### 7.3 疑問文の形成

疑問文は文末に疑問標識「ka」を付加します。疑問詞疑問文では、疑問詞は通常の位置に置かれます。

### 7.4 複文構造

複文は様々な接続詞を用いて形成されます。主要な接続詞には以下のものがあります：
- dak：理由
- kas：逆接
- nod：原因
- soʃ：並列

## 第11部：語彙拡張メカニズム

### 11.1 複合語形成

**複合パターン：**
1. 直接結合型：N₁ + N₂ → N₁N₂
2. 属格型：N₁ + no + N₂ → N₁nN₂
3. 動詞語幹型：V-幹 + N → VN

**複合時の母音調和：**
- 主要部優先規則：後部要素の母音タイプが複合語全体を支配
- 境界弱化：複合語境界で母音調和制約が弱まる場合あり

### 11.2 派生形態論

**接頭辞による派生：**
- spr-/spir-：超越・卓越
- an-/en-：対立・反対
- ml-/mil-：多様・複数

**接尾辞による派生：**
- -loʒ/-leʒ：学問分野
- -tik/-tak：性質・特性

### 11.3 借用語の適応

**音韻適応規則：**
- 子音クラスタの分解
- 末尾子音の処理
- 非許容音素の代替

**借用層による適応差：**
- 古層：完全な母音調和適応
- 中層：部分的母音調和
- 新層：音韻構造の部分保持
- 最新層：最小限の適応

## 付録B：辞書データベース構造

### B.1 データベース概要

ニンゴト語辞書は関係データベースとして実装されており、様々な表（テーブル）が相互に関連付けられています。主要なテーブルは以下の通りです：

- **stems**：語幹（見出し語）の基本情報
- **phonological_patterns**：音韻パターン情報（7種類）
- **parts_of_speech**：品詞情報（11種類）
- **inflection_types**：曲用・活用の種類の定義
- **inflections**：個別の語の実際の曲用・活用形
- **definitions**：語義情報
- **sentences**：例文情報

### B.2 音韻パターン分類

データベース内の音韻パターンは以下の7種類に分類されています：

1. 前舌母音語幹：前舌母音（i, e, æ）で終わる語幹
2. 後舌母音語幹：後舌母音（o, u, ɔ）で終わる語幹
3. 子音語幹：子音で終わる語幹
4. ə語幹：əで終わる語幹
5. a語幹：aで終わる語幹
6. 前舌子音語幹：前舌母音調和を持つ子音で終わる語幹
7. 後舌子音語幹：後舌母音調和を持つ子音で終わる語幹

### B.3 変化形のカテゴリ構造

#### B.3.1 名詞曲用カテゴリ（32種類）

名詞曲用は以下の文法素性の組み合わせによって32種類の変化形を生成します：

- 数：単数／複数（2種）
- 定性：定／不定（2種）
- 格：主格、対格、与格、所格、属格、出格、向格、共格（8種）

これらの組み合わせ：2 × 2 × 8 = 32種類

#### B.3.2 形容詞曲用カテゴリ（64種類）

形容詞曲用は名詞の曲用カテゴリに有生性の区別が加わり、64種類の変化形を生成します：

- 数：単数／複数（2種）
- 定性：定／不定（2種）
- 格：主格、対格、与格、所格、属格、出格、向格、共格（8種）
- 有生性：有生／無生（2種）

これらの組み合わせ：2 × 2 × 8 × 2 = 64種類

#### B.3.3 動詞活用カテゴリ（直説法108種類）

直説法の動詞活用は以下の文法素性の組み合わせによって108種類の変化形を生成します：

- 時制：現在、過去、未来（3種）
- 相：非完了相、完遂相、結果状態相、準備相、起動相、終結相（6種）
- 人称：一人称、二人称、三人称（3種）
- 数：単数、複数（2種）

これらの組み合わせ：3 × 6 × 3 × 2 = 108種類

### B.4 データ検索の例

辞書データベースから名詞「mir」（人）の単数定主格形を検索する場合：

```sql
SELECT i.form 
FROM inflections i 
JOIN inflection_types it ON i.inflection_type_id = it.id 
JOIN stems s ON i.stem_id = s.id 
WHERE s.stem = 'mir' AND it.name = '単数定主格';
```

同様に、動詞「kur」（来る）の直説法現在時制非完了相三人称単数形を検索する場合：

```sql
SELECT i.form 
FROM inflections i 
JOIN inflection_types it ON i.inflection_type_id = it.id 
JOIN stems s ON i.stem_id = s.id 
WHERE s.stem = 'kur' AND it.name = '直説法現在時制非完了相三人称単数';
```
