# ニンゴト語ブックレット(第4版)
## 音韻体系から文法構造まで

---

## 序論

ニンゴト語は一般的な日本語から1000年後に発達した言語とされていますが、本ブックレットではその言語学的特徴を体系的に記述します。本書は、ニンゴト語の文法体系を包括的に説明する目的で編纂されています。特に、データベースに格納された曲用・活用パターンと実際の言語使用の関係に焦点を当てています。

### 1.3 音節構造

基本的な音節構造は CVC(C) であり、語末に限り子音連続 CC が許容されます。また、/r̃/, /l̃/, /ñ/, /m̃/ の4つの子音は、音節主音としての機能を持ち、母音の代わりに音節核となることができます。

#### 鼻音連続の縮約

音節主音の鼻音が同種の鼻音と隣接する場合、CVC(C) 制約を維持するために縮約が起きます：

- ñn, ñm → ñ
- m̃m, m̃n → m̃

例：「本に」hon + n → hñ + ñ → hñ（ñn が縮約）

### 1.4 母音調和

ニンゴト語には母音調和の原理が存在し、単語内の母音は以下の3群に分類されます：

| グループ | 母音 | 特徴 |
| -------- | ---- | ---- |
| 前舌 | i, e, æ | 調和グループ |
| 後舌 | u, o, ɔ | 調和グループ |
| 中立 | ə, a /ä/ | どちらの群とも共存可能 |

a は IPA /ä/（中舌寄りの広母音）であり、ə と同様に中立母音として扱われます。母音調和は接辞にも適用され、語幹の母音群に応じて適切な形態が選択されます。

動詞の連結母音は歴史的に全て中立母音 a でしたが、後舌語幹の環境では母音調和の強化により a → ɔ に同化しました。前舌語幹では a が中立のため変化を受けず保持されています。

## 第2部：形態統語論的特徴

### 2.0 類型論的位置づけ

ニンゴト語は膠着語（現代日本語）を祖語とし、屈折語への移行途上にある混合型の言語です。文法領域ごとに形態論的タイプが異なり、歴史的地層を反映しています：

| 文法領域 | 形態論的タイプ | 特徴 |
| -------- | -------------- | ---- |
| 名詞・形容詞 | 屈折語的 | 格+数の融合、形容詞の一致変化 |
| 動詞 TAP（時制・相・人称） | 融合進行中 | 古層は融合的（語幹=時制）、新層は膠着的（人称接辞） |
| 動詞の法 | 分析的 | 接頭辞として ~ で結合。融合に至っていない |
| 定性 | 分析的 | 不変化前接語 sə |

この屈折語化は、逆説的に、法的表現の文法化によって可能になりました。現代日本語は厳密には「非過去（〜る）/ 完了・実現（〜た）」の二項対立を持つのみで、正確に時制と呼べるカテゴリーは現在しか存在しません。ニンゴト語の3時制体系は以下の経路で成立しました：

1. **非過去 → 現在**：未来の意味が分離し、「今」に限定
2. **〜た → 過去**：完了・実現（realis）から真の時制（過去）に再分析
3. **だろう → 未来**：推量法が時制に文法化（→ dəro-）

同様に、人称代名詞の縮約が人称接辞を生み出しました（vəç → -w, karl → -r 等）。つまり法的表現と人称代名詞が「時制」「人称」という文法スロットに流入したことで、動詞活用に屈折的パラダイムの土台が形成されました。

一方で、まだ意味が具体的なままの法表現（bək「べき」、kəm「かもしれない」等）は、活用パラダイムに取り込まれるには9種×既存カテゴリで組み合わせが爆発的に増加するため（3時制 × 6相 × 6人称数 × 9法 = 972形/動詞）、分析的な接頭辞として ~ で結合する段階に留まっています。将来的には高頻度の法（命令 ja-、義務 bək- 等）から融合が始まる可能性があります。

## 第3部：名詞の曲用体系

### 3.1 名詞分類

名詞は以下の基準で分類されます：

1. **母音調和による分類**
   - 前舌母音系名詞（i, e, æを含む語幹）
   - 後舌母音系名詞（o, u, ɔを含む語幹）
   - a語幹名詞（前舌でも後舌でもなく、aを含む語幹）
   - ə語幹名詞（前舌でも後舌でもなく、əを含む語幹）

2. **文法性による分類**
   - 有生名詞
   - 無生名詞

3. **数による分類**
   - 単数
   - 複数

4. **定性による分類**
   - 不定
   - 定

### 3.2 格体系

ニンゴト語は8つの基本格を持ちます：

| 格   | 機能             |
| ---- | ---------------- |
| 主格 | 動作・状態の主体 |
| 対格 | 動作の対象       |
| 与格 | 間接目的語       |
| 所格 | 場所・手段       |
| 属格 | 所有・関係       |
| 出格 | 起点・比較       |
| 向格 | 方向・目標       |
| 共格 | 共同・相互関係   |

### 3.3 曲用パターン

曲用接尾辞の順序は以下の通りです：
**語幹 + 格標識 + 複数標識 + 定標識**

一般的には「格標識 + 複数標識 + 定標識」が融合した曲用語尾が使用されます。

#### 3.3.1 前舌母音系名詞

**有生名詞の曲用パターン**

- 単数/複数区分
- 定/不定区分
- 複数形成時の音韻調整：一部の格では接続母音を挿入

**無生名詞の曲用パターン**

- 基本的に有生名詞と同じ曲用を示すが、複数形での属格形成に差異がある

#### 3.3.2 後舌母音系名詞

**有生名詞の曲用パターン**

- 前舌系と同様の基本構造
- 対応する母音調和形を使用
- 一部の格では独自の形態を持つ

**無生名詞の曲用パターン**

- 複数形での所格、出格、向格、共格に接続母音「-a」を挿入

## 第4部：形容詞の曲用体系

### 4.1 形容詞分類

形容詞は名詞と同様に以下のように分類されます：

1. **母音調和による分類**
   - 前舌母音系形容詞
   - 後舌母音系形容詞

2. **修飾対象による分類**
   - 有生名詞修飾形
   - 無生名詞修飾形

### 4.2 形容詞の一致

形容詞は修飾する名詞と以下の特性において一致します：

- 数（単数/複数）
- 定性（定/不定）
- 格
- 有生性

### 4.3 曲用パターン

形容詞の曲用は基本的に名詞の曲用パターンに従いますが、修飾対象によって異なる形態をとることがあります。

#### 4.3.1 前舌母音系形容詞

**有生名詞修飾形**
- 修飾対象の名詞と完全に一致する曲用を示す

**無生名詞修飾形**
- 複数属格形成において名詞と異なるパターンを示す

#### 4.3.2 後舌母音系形容詞

**有生名詞修飾形**
- 修飾対象の名詞と完全に一致する曲用を示す

**無生名詞修飾形**
- 基本的に有生名詞修飾形と同じだが、接続母音に差異がある場合がある

### 4.4 比較級と最上級

#### 4.4.1 比較級

比較接辞 **-jər** を形容詞語幹に付加します。-jər は「より」/jori/ の弱化形で、ə が中立母音のため前舌・後舌どちらの形容詞にも付きます。比較級形は子音語幹の形容詞として曲用し、修飾する名詞の格・数・有生性に一致します。

比較対象は出格で標示します：

- ɔkjəra sə kii-ukr̃「あの木より大きい」（ɔk + jər + nom.sg、kii + abl）
- çisjəri sə çit-ikr̃「あの人より小さい」

#### 4.4.2 最上級

定冠詞 sə + 比較級で最上級を表します（ロマンス語型の分析的構成）：

- sə ɔkjəra「最も大きい」
- sə çisjəri「最も小さい」

専用の最上級接辞は存在せず、定冠詞が「その集合の中で」という限定を加えることで最上級の意味を生じさせます。

### 4.5 数詞
数詞は形容詞的用法を持ちますが、不変化詞として扱われます。数詞は名詞の前に置かれますが、名詞の曲用に一致せず、曲用形も持ちません。

#### 4.4.1 基本的な数詞
数詞はすべて前舌調和します。

| 数字  | 数詞  |
| :---: | :---: |
|   1   |  çti  |
|   2   |  fn   |
|   3   |  sn   |
|   4   |  ʃi   |
|   5   |  ge   |
|   6   |  rek  |
|   7   |  neŋ  |
|   8   |  ça   |
|   9   |  ki   |
|  10   |  ʒi   |
|  100  |  jak  |
| 1000  |  ʃəŋ  |
| 10000 |  maŋ  |

11以上の数詞は、基本的な数詞を組み合わせて形成されます。例えば、21は「fnʒi çti」となります。

例；
- snjak fnʒi rek (326)
- ʃiʒi neŋ (47)
- çti maŋ rekʃəŋ snjak geʒi çti (16351)

上記の例からもわかるように、一万以上の数は日本語と同様に万進法に従います。

- maŋ (1万)
- ek (1億)
- tʃe (1兆)
- kej (1京)

#### 4.4.2 基数詞
基数詞は名詞の前に置かれ、名詞の曲用に一致しません。基数詞は形容詞的用法を持ちます。
基数詞は単純に数詞に「-i」(母音末尾にあっては「-mi」)を付加することで形成されます。

| 数字  | 数詞  |
| :---: | :---: |
|   1   | çtimi |
|   2   |  fni  |
|   3   |  sni  |
|   4   | ʃimi  |
|   5   | gemi  |
|   6   | reki  |
|   7   | neŋi  |
|   8   | çami  |
|   9   | kimi  |
|  10   | ʒimi  |

#### 4.4.3 単位付きの数詞
単位付きの数詞は、単位を名詞として扱うことで実現されます。

- metr̃ (メートル)
- kiləmetr̃ (キロメートル)
- gram (グラム)
- kiləgram (キログラム)

通常の名詞と同様に、数詞を単に名詞の前に置くことで修飾します。

- çti metr̃i (1メートル, 主格)

また、他の名詞を修飾する場合は、単位を属格とします。

- çti metr̃me kii (1メートルの木; metr̃を属格とし、kiiの数と定性に一致させる)

## 第5部：代名詞体系

### 5.1 人称代名詞

人称代名詞は、一般の名詞とは異なる独自の曲用パターンを持ちます。人称代名詞は所格を欠きます。

基本形式：
- 1人称単数：vəç
- 2人称単数：ant
- 3人称単数：kar
- 1人称複数：vəçl
- 2人称複数：antes
- 3人称複数：karl

人称代名詞は場所や時間を表すことがないため、所格は存在しません。

### 5.2 指示代名詞

指示代名詞は距離の3段階区分を持ちます：
- 近称：ko- 系統
- 中称：so- 系統
- 遠称：a- 系統

また、疑問・不定形式として do- 系統があります。

| 種類           | 主格  | 対格  | 与格  | 属格   | 所格  | 出格   | 向格  | 共格  |
| -------------- | ----- | ----- | ----- | ------ | ----- | ------ | ----- | ----- |
| 近称単数       | kori  | kore  | korñ  | korne  | kok   | kokr   | kokñ  | kort  |
| 近称複数       | korir | korer | korñr | korner | kokid | kokikr̃ | kokiñ | korit |
| 中称単数       | sori  | sore  | sorñ  | sorne  | sok   | sokr   | sokñ  | sork  |
| 中称複数       | sorir | sorer | sorñr | sorner | sokid | sokikr̃ | sokiñ | sorit |
| 遠称単数       | ari   | are   | arñ   | arne   | akr   | akr    | akñ   | akt   |
| 遠称複数       | arir  | arer  | arñr  | arner  | akrad | akrakr̃ | akñiñ | aktit |
| 疑問・不定単数 | dori  | dore  | dorñ  | dorne  | dok   | dokr   | dokñ  | dort  |
| 疑問・不定複数 | dorir | dorer | dorñr | dorner | dokid | dokikr̃ | dokiñ | dorit |

## 第6部：動詞の活用体系

### 6.1 動詞分類

動詞は以下のように分類されます：

1. **母音調和による分類**
   - 前舌母音系動詞
   - 後舌母音系動詞

2. **活用の規則性による分類**
   - 規則動詞
   - 不規則動詞

### 6.2 動詞の構造

動詞は以下の構造を持ちます：
**接頭辞 + 法接頭辞 + 語幹 + 敬語接辞 + 活用語尾 + 態標識 + 否定接尾辞**

### 6.3 接頭辞体系

様々な文法的・意味的機能を持つ接頭辞があり、動詞語幹の前に付加されます。

### 6.4 活用語尾

活用語尾は時制・アスペクト・人称・数を表現します。歴史的な地層構造を持ちます：

- **最古層（融合的）**：語幹+時制。コピュラ ar 等で語幹が時制を内包する形で残存
- **古層（部分融合）**：アスペクト接辞+時制標識。過去時制との融合が見られる（sm+ta→sma, tr+ta→tra, tk+ta→kta, ds+ta→sta）
- **新層（膠着的）**：人称標識。代名詞の縮約から発達
- **最新層（語彙化）**：法。接頭辞として動詞に結合（6.5節参照）

#### 6.4.1 人称標識

| 人称  | 単数     | 複数 | 由来                 |
| ----- | -------- | ---- | -------------------- |
| 1人称 | -w       | -l   | vəç / vəçl           |
| 2人称 | 母音上昇 | -s   | 対面発話変異 / antes |
| 3人称 | ∅        | -r   | kar 脱落 / karl      |

2人称単数は語尾の時制母音を上昇させることで標示します：

- **前舌調和**: e→i, a→e（例：mire → miri）
- **後舌調和**: u→uj（音割れ）, ɔ→o（例：jomu → jomuj）
- **未来時制**: 上昇の代わりに子音 -t を付加（例：miræ → miræt）

#### 6.4.2 アスペクト接辞

6種類のアスペクトを接辞で標示します。連結母音（前舌 a、後舌 ɔ）を介して語幹に接続します：

| アスペクト | 接辞 | 過去融合形 | 意味 | 語源 |
| ---------- | ---- | ---------- | ---- | ---- |
| 非完了相 | (なし) | — | 進行・習慣 | — |
| 完遂相 | -sm- | sm+ta→sma | 完了・達成 | 「しまう」/ʃimau/ → /sm/ |
| 結果状態相 | -tr- | tr+ta→tra | 結果の持続 | 「ている」/teiru/ → /tr/ |
| 準備相 | -tk- | tk+ta→kta | 準備段階 | 「ておく」/teoku/ → /tk/ |
| 起動相 | -ds- | ds+ta→sta | 開始 | 「だす」/dasu/ → /ds/ |
| 終結相 | -v- | v+ta→vta | 終了・完了 | 「おえる」/oeru/ → /ov/ → /v/ |

全て日本語の「〜して＋補助動詞」構文に由来します。補助動詞の子音骨格のみが残り、「して」の部分は脱落しました。母音語幹動詞では語幹末母音が連結母音の役割を果たすため、これらの接辞は語幹末母音に直接後続します。

### 6.5 法（モダリティ）接頭辞

| 現代日本語表現     | 接頭辞   | 法         | 例                               |
| ------------------ | -------- | ---------- | -------------------------------- |
| 〜                 | (ゼロ)   | 直説法     | 事実の叙述                       |
| 〜なければならない | **bək-** | 義務法     | **bək**-ik-a (行かねばならない)  |
| 〜かもしれない     | **kəm-** | 可能法     | **kəm**-fal-a (降るかもしれない) |
| 〜はずだ           | **həz-** | 推定法     | **həz**-kur-a (来るはずだ)       |
| 〜たがる           | **tag-** | 希求法     | **tag**-jom-a (読みたがる)       |
| 〜そうだ（様態）   | **səj-** | 様態推量法 | **səj**-ɔk-a (大きそうだ)        |
| 〜そうだ（伝聞）   | **sak-** | 伝聞法     | **sak**-kur-a (来るそうだ)       |
| 〜であれば         | **rə-**  | 条件法     | **rə**-kur-a (来たならば)        |
| 〜しろ             | **ja-**  | 命令法     | **ja**-kur-a (来い)              |

### 6.6 否定

否定は接尾辞「-ñ」で表現されます。この接尾辞は人称標識の後に付加されます（例：1sg -wñ, 1pl -lñ, 3pl -rñ）。

日常体では**鼻母音化規則**が適用されます：否定標識 -ñ の前の時制母音が鼻母音化し、-ñ 自体は脱落します。

- mirewñ → mirẽw（見ない・1sg）
- mekẽ（行かない・3sg）
- jomuñ → jomũ（読まない・3sg）

強調体では -ñ が保存され、鼻母音化は起きません。

### 6.7 準動詞

ニンゴト語には以下の準動詞形式があります：
- *-kot/-ket: 名詞化（後舌/前舌、母音語末形）
- *-əkot/-əket: 名詞化（後舌/前舌、子音語末形）

準動詞は名詞的用法と形容詞的用法を持ち、それぞれの機能に応じた曲用を示します。

## 第7部：統語構造

### 7.1 基本語順

ニンゴト語は主要部後置型のSOV言語です：
**主語 + 目的語 + 動詞**

### 7.2 関係節構造

関係節は動詞に「-de/-do」という関係節標識を付加して形成します：
**関係節 + 被修飾名詞**

関係節内で修飾される名詞は欠落（ギャップ戦略）します。

### 7.3 疑問文の形成

疑問文は文末に疑問標識「ka」を付加します。疑問詞疑問文では、疑問詞は通常の位置に置かれます。

### 7.4 複文構造

複文は様々な接続詞を用いて形成されます。主要な接続詞には以下のものがあります：
- dak：理由
- kas：逆接
- nod：原因
- soʃ：並列

## 付録A：特殊変化と不規則形

### A.1 不規則動詞

一部の高頻度動詞は不規則な活用パターンを示します。

#### コピュラ ar（存在する・〜である）

3語幹の統合による補充形動詞。Lyons の4機能（存在・同定・所在・属性付与）をカバーします。

**語幹分布**：1人称→dər、3人称有生→ir、未来全域→dər、それ以外→ar/er

**主な不規則性**：
- 時制母音を持たず、語幹自体が時制を内包（最古層の融合的パターン）
- 過去時制で r→t 交替（ar→at, ir→it, dər→dət）
- アスペクトは非完了相と終結相のみ

**新人称体系への適応**：

コピュラは裸語幹のため、2sg の母音上昇を適用する時制母音がなく、2sg と 3sg が衝突しました。この衝突は二つの異なるメカニズムで解消されています：

- **現在/過去**：3sg（無生）が ir/it との類推で a→e correction を受け、er/et に変化。2sg は対面発話の高頻度で原形 ar/at を保持。
- **未来**：2sg の使用頻度が低く、correction を受ける側に。弱母音 ə が u に置換（dəro→duro）。

| 時制 | 1sg   | 2sg  | 3sg(無生) | 3sg(有生) | 1pl   | 2pl   | 3pl(無生) | 3pl(有生) |
| ---- | ----- | ---- | --------- | --------- | ----- | ----- | --------- | --------- |
| 現在 | dərw  | ar   | er        | ir        | dərl  | ars   | err       | irr       |
| 過去 | dətw  | at   | et        | it        | dətl  | ats   | etr       | itr       |
| 未来 | dərow | duro | dəro      | dəro      | dərol | dəros | dəror     | dəror     |

#### 移動動詞 ik（行く）

3語幹の統合：ik（←行く）、dər（←出る）、mek（←向かう）。語幹選択を除き verb_front_cons クラスに委譲するため、人称標識は規則的に適用されます。

**語幹分布**：1sg→ik、2人称現在/過去→dər、それ以外→mek

## 付録B：辞書データベース構造

### B.1 データベース概要

ニンゴト語辞書は関係データベースとして実装されており、様々な表（テーブル）が相互に関連付けられています。主要なテーブルは以下の通りです：

- **stems**：語幹（見出し語）の基本情報
- **phonological_patterns**：音韻パターン情報（7種類）
- **parts_of_speech**：品詞情報（11種類）
- **inflection_types**：曲用・活用の種類の定義
- **inflections**：個別の語の実際の曲用・活用形
- **definitions**：語義情報
- **sentences**：例文情報

### B.2 音韻パターン分類

データベース内の音韻パターンは以下の7種類に分類されています：

1. 前舌母音語幹：前舌母音（i, e, æ）で終わる語幹
2. 後舌母音語幹：後舌母音（o, u, ɔ）で終わる語幹
3. 子音語幹：子音で終わる語幹
4. ə語幹：əで終わる語幹
5. a語幹：aで終わる語幹
6. 前舌子音語幹：前舌母音調和を持つ子音で終わる語幹
7. 後舌子音語幹：後舌母音調和を持つ子音で終わる語幹

### B.3 変化形のカテゴリ構造

#### B.3.1 名詞曲用カテゴリ（32種類）

名詞曲用は以下の文法素性の組み合わせによって32種類の変化形を生成します：

- 数：単数／複数（2種）
- 定性：定／不定（2種）
- 格：主格、対格、与格、所格、属格、出格、向格、共格（8種）

これらの組み合わせ：2 × 2 × 8 = 32種類

#### B.3.2 形容詞曲用カテゴリ（64種類）

形容詞曲用は名詞の曲用カテゴリに有生性の区別が加わり、64種類の変化形を生成します：

- 数：単数／複数（2種）
- 定性：定／不定（2種）
- 格：主格、対格、与格、所格、属格、出格、向格、共格（8種）
- 有生性：有生／無生（2種）

これらの組み合わせ：2 × 2 × 8 × 2 = 64種類

#### B.3.3 動詞活用カテゴリ（直説法108種類）

直説法の動詞活用は以下の文法素性の組み合わせによって108種類の変化形を生成します：

- 時制：現在、過去、未来（3種）
- 相：非完了相、完遂相、結果状態相、準備相、起動相、終結相（6種）
- 人称：一人称、二人称、三人称（3種）
- 数：単数、複数（2種）

これらの組み合わせ：3 × 6 × 3 × 2 = 108種類

### B.4 データ検索の例

辞書データベースから名詞「mir」（人）の単数定主格形を検索する場合：

```sql
SELECT i.form 
FROM inflections i 
JOIN inflection_types it ON i.inflection_type_id = it.id 
JOIN stems s ON i.stem_id = s.id 
WHERE s.stem = 'mir' AND it.name = '単数定主格';
```

同様に、動詞「kur」（来る）の直説法現在時制非完了相三人称単数形を検索する場合：

```sql
SELECT i.form 
FROM inflections i 
JOIN inflection_types it ON i.inflection_type_id = it.id 
JOIN stems s ON i.stem_id = s.id 
WHERE s.stem = 'kur' AND it.name = '直説法現在時制非完了相三人称単数';
```
