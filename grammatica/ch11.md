# Grammatica Nuçongə — Capitulum XI

## 1. 向格（方向の格）

向格は**方向・目標**を表します。「〜へ」「〜に向かって」に相当します。

### 1.1 向格単数

向格語尾の核は **-h** です。

| 語幹タイプ | 語尾 | 例 |
|-----------|------|-----|
| 前舌子音 | **-he** | çit → **çithe**, Lin → **Linhe** |
| 後舌子音 | **-uh** | tokl → **tokluh**, Tak → **Takuh** |
| 前舌母音 | **-he** | ki → **kihe** |
| 中立子音 | **-hə** | sakən → **sakənhə** |
| æ (不規則) | — | **æihə** |

代名詞の向格：

| 代名詞 | 向格sg |
|--------|--------|
| vəç | **vəçeñ** |
| ant | **anteñ** |
| kar | **karñ** |

> Taka **tokluh** meke. (Takは場所へ行く。)
> Lini **æihə** kuru. (Linは家へ来る。)

---

## 2. 出格 + 向格: 完全な移動表現

出格（起点）と向格（目標）を組み合わせて、出発地と目的地を明示できます。

```
主語(主格) + 起点(出格) + 目標(向格) + 移動動詞
```

> Taka **æikr̃** **tokluh** meke. (Takは家から場所へ行く。)
> Lini **toklukr̃** **æihə** kuru. (Linは場所から家へ来る。)

---

## 3. 空間三格の対比

| 格 | 機能 | æ の形 | 例文 |
|----|------|--------|------|
| **所格** | 場所にいる | **æid** | Taka æid ir. (家にいる) |
| **向格** | 場所へ向かう | **æihə** | Taka æihə meke. (家へ行く) |
| **出格** | 場所から離れる | **æikr̃** | Taka æikr̃ meke. (家から行く) |
