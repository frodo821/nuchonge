# Grammatica Nuçongə — Capitulum IV

## 1. 対格複数

複数の目的語を表すには対格複数形を用います。

### 1.1 複数標識

複数形には語幹と格語尾の間に**複数標識**が挿入されます。有生・無生で標識が異なります：

| 調和 | 有生 | 無生 |
|------|------|------|
| 前舌 | **-ir-** / **-r-** | **-er-** / **-r-** |
| 後舌 | **-ar-** / **-r-** | **-or-** |
| 中立 | **-ər-** / **-r-** | **-ər-** |

### 1.2 対格複数の形

| 語幹タイプ | 対格sg | 対格pl | 例 |
|-----------|--------|--------|-----|
| 前舌子音・有生 | -e | **-re** | çite → **çitre** |
| 前舌子音・無生 | -e | **-ere** | hine → **hinere** |
| 後舌子音・有生 | -o | **-ro** | tolo → **tolro** |
| 後舌子音・無生 | -o | **-oro** | hño → **hñoro** |
| 前舌母音・無生 | -je | **-re** | kije → **kire**, mije → **mire** |
| 中立子音・有生 | -a | **-ra** | sakəna → **sakənra** |

> Taka **hñoro** jomu. (Takは本々を読む。)
> Lini **kire** mire. (Linは木々を見る。)
> Taka **tolro** mire. (Takは鳥たちを見る。)
> Lini **sakənra** mire. (Linは魚たちを見る。)

## 2. 有生と無生の複数形対比

有生名詞と無生名詞では複数標識が異なるため、同じ格でも形が違います。

### 前舌子音語幹の比較

| | 有生 (çit 人) | 無生 (hin 品) |
|--|-------------|-------------|
| 主格sg | çit**i** | hin**i** |
| 主格pl | çit**ri** | hin**eri** |
| 対格sg | çit**e** | hin**e** |
| 対格pl | çit**re** | hin**ere** |

### 後舌子音語幹の比較

| | 有生 (tol 鳥) | 無生 (hñ 本) |
|--|-------------|-------------|
| 主格sg | tol**a** | hñ**a** |
| 主格pl | tol**ra** | hñ**ora** |
| 対格sg | tol**o** | hñ**o** |
| 対格pl | tol**ro** | hñ**oro** |

有生名詞は複数標識が短く（-r-/-ir-/-ar-）、無生名詞はやや長い（-er-/-or-/-ər-）のが特徴です。

## 3. 単数と複数の対比

| 名詞 | 対格sg | 対格pl |
|------|--------|--------|
| hñ (本) | hñ**o** | hñ**oro** |
| ki (木) | ki**je** | ki**re** |
| mi (実) | mi**je** | mi**re** |
| tol (鳥) | tol**o** | tol**ro** |
| sakən (魚) | sakən**a** | sakən**ra** |
| çit (人) | çit**e** | çit**re** |

> Taka tol**o** mire. (Takは鳥を見る。)
> Taka tol**ro** mire. (Takは鳥たちを見る。)
