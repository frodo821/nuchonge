# Grammatica Nuçongə — Capitulum XVIII

## 1. 比較級の形成

比較級は形容詞の語幹に **-jər** を付加して形成します：

| 形容詞 | 原級語幹 | 比較級語幹 |
|--------|---------|-----------|
| ɔk（大きい） | ɔk- | **ɔkjər**- |
| çis（小さい） | çis- | **çisjər**- |
| alaj（あらゆる） | alaj- | **alajjər**- |

比較級語幹は原級と同じリンカー融合曲用で格変化します。

### 1.1 比較級のパラダイム（ɔkjər, 後舌, 無生sg）

| 格 | 原級 | 比較級 |
|----|------|--------|
| 主格 | ɔkaŋ | ɔkjər**aŋ** |
| 対格 | ɔkoŋ | ɔkjər**oŋ** |
| 属格 | ɔkmoŋ | ɔkjər**moŋ** |
| 所格 | ɔkdoŋ | ɔkjər**doŋ** |

### 1.2 前舌（çisjər, 無生sg）

| 格 | 原級 | 比較級 |
|----|------|--------|
| 主格 | çisiŋ | çisjər**iŋ** |
| 対格 | çiseŋ | çisjər**eŋ** |
| 属格 | çismeŋ | çisjər**meŋ** |

---

## 2. 出格による比較対象

比較の基準（「〜より」）は**出格**で表します：

```
比較基準[出格] + 主語 + 形容詞[比較級] + コピュラ
```

> **Linne æikr̃** Takno æ ɔkjəraŋ er.
> （**Linの家より** Takの家はもっと大きい。）

> **Linikr̃** Taka ɔkjəraŋ ir.
> （**Linより** Takは大きい。）

> **Takikr̃** Lini çisjəriŋ ir.
> （**Takより** Linは小さい。）

出格は第10章で学んだ起点の格ですが、ここでは比較の起点として使います。これは「〜から離れて（もっと大きい方へ）」という空間的比喩に基づきます。

---

## 3. 比較級の限定用法

比較級も原級と同じく名詞の前に置いて限定用法で使えます：

> Taka **ɔkjəroŋ** hño jomu. （Takは**もっと大きい**本を読む。）
> Lini **çisjəreŋ** mije tæbene. （Linは**もっと小さい**実を食べる。）

---

## 4. 最上級: sə + 比較級

最上級は定冠詞 **sə** を比較級の前に置いて分析的に形成します：

| | 形式 | 意味 |
|---|------|------|
| 原級 | ɔkaŋ | 大きい |
| 比較級 | ɔkjəraŋ | もっと大きい |
| 最上級 | **sə** ɔkjəraŋ | **一番**大きい |

> **sə ɔkjəraŋ** æ er. （**一番大きい**家がある。）
> Taka **s ɔkjəroŋ** hño jomu. （Takは**一番大きい**本を読む。）

sə は第1章で導入された定冠詞です。「定まった比較級」→「最も〜な」という意味拡張です。

---

## 5. 度合いの体系まとめ

| 度合い | 形成法 | 例 |
|--------|--------|-----|
| 原級 | 語幹 + リンカー曲用 | ɔkaŋ |
| 比較級 | 語幹 + **jər** + リンカー曲用 | ɔkjəraŋ |
| 最上級 | **sə** + 比較級 | sə ɔkjəraŋ |
