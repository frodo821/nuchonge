# Grammatica Nuçongə — Capitulum XV

## 1. アスペクト（相）の導入

これまでの章では、すべての動詞が**非完結相** (impf) で活用されていました。非完結相は動作の進行・反復・習慣を表し、特別な標識を持ちません。

ヌヒョン語には6種のアスペクトがあります：

| アスペクト | 標識 | 意味 |
|-----------|------|------|
| 非完結 (impf) | ∅ | 進行・反復・状態 |
| **終結 (term)** | **-v-** | **動作の完了点到達** |
| 結果状態 (result) | -tr- | 完了後の状態持続 |
| 準備 (prep) | -tk- | 動作の事前準備 |
| 起動 (incep) | -ds- | 動作の開始 |
| 完遂 (perf) | -sm- | 動作の完全達成 |

この章では**終結相 (term)** を学びます。終結相は動作が**自然な終点に達すること**を表し、日本語の「〜し終える」「〜し終わる」に相当します。

---

## 2. 終結相の形成

終結相は語幹と時制母音の間に **-v-** を挿入して形成します。

### 基本構造

```
語幹 + (連結母音) + v + 時制母音 + 人称接辞
```

非完結相との比較：

| | impf | term |
|---|------|------|
| 構造 | 語幹 + 時制母音 + 人称 | 語幹 + **v** + 時制母音 + 人称 |
| 例 (jom) | jom-u | jom-**ɔv**-u |
| 例 (mir) | mir-e | mir-**av**-e |

## 5. impf と term の意味対比

| impf | term | 違い |
|------|------|------|
| Taka hño **jomu**. | Taka hño **jomɔvu**. | |
| Takは本を読んでいる | Takは本を読み終える | 進行 → 完了点 |
| Lini mije **tæbene**. | Lini mije **tæbenave**. | |
| Linは実を食べている | Linは実を食べ終える | 進行 → 完了点 |

終結相は動作の**自然な終点**に焦点を当てます。「本を読む」動作には「最後のページまで読む」という自然な終点があり、終結相はその到達を表します。

## 6. 終結相の否定

否定は非完結相と同じ鼻母音化規則で形成します。時制母音が鼻母音化します：

| 時制 | 肯定 | 否定 |
|------|------|------|
| 現在 (後舌) | jomɔvu | jomɔv**ũ** |
| 現在 (前舌) | mirave | mirav**ẽ** |
| 過去 (後舌) | jomɔvtɔ | jomɔvt**ɔ̃** |
| 過去 (前舌) | miravta | miravt**ã** |

> Taka hño jomɔvũ. (Takは本を読み終えない。)
> Lini kije miravẽ. (Linは木を見終えない。)

## 7. 副詞 ma と mar

| 副詞 | 意味 | 典型的な相との共起 |
|------|------|-------------------|
| **ma** | もう（完了・既遂） | 終結相と共起しやすい |
| **mar** | まだ（継続・未完了） | 非完結相と共起しやすい |

ma と mar は動詞の直前に置かれます：

> Taka hño **mar** jomu. (Takは**まだ**本を読んでいる。)
> Taka hño **ma** jomɔvu. (Takは**もう**本を読み終える。)

ma + 終結相 = 「もう〜し終えた/終える」
mar + 非完結相 = 「まだ〜している」

この対比は動作の進行と完了の違いを明確に表現します。
