# Grammatica Nuçongə — 文法解説全22章

`textbook/` の `.hut` 各章と対をなす、ヌヒョン語の文法解説集です。各章は `textbook/chNN.hut` に現れる新出文法事項を日本語で解説します。全22章は3部構成で、格体系 → 時制・人称 → アスペクト・修飾 の順に積み上がります。

包括的な文法記述は [`../booklet_4th`](../booklet_4th) (ブックレット第4版) を参照。本章群は学習者向けの段階的導入、ブックレットは体系的リファレンスという役割分担です。

---

## 第I部：格体系の基礎（第1–7章）

名詞の8格のうち主要7格と、コピュラ・定冠詞・形容詞述語用法を導入。

- [第1章](./ch01) — コピュラ `ar`、主格、定冠詞 `sə`、形容詞述語用法
- [第2章](./ch02) — 対格、他動詞活用 (3sg 現在非完結)、SOV 語順
- [第3章](./ch03) — 属格、3人称代名詞 `kar`、人物名の曲用
- [第4章](./ch04) — 対格複数、有生/無生の複数対比
- [第5章](./ch05) — 1・2人称代名詞、動詞の人称標識
- [第6章](./ch06) — 与格、授与動詞 `atɔw`
- [第7章](./ch07) — 所格・共格、**第I部の格体系総覧**

## 第II部：時制・人称・格体系の完成（第8–14章）

3時制と複数人称、残り2格（出格・向格）、否定、指示詞。

- [第8章](./ch08) — 過去時制、時間副詞、pro-drop
- [第9章](./ch09) — 未来時制 (`dəro-`)、3時制の総覧
- [第10章](./ch10) — 出格、移動動詞 `kur`／`ik` (補充形)
- [第11章](./ch11) — 向格、空間三格の対比、**8格体系の完成**
- [第12章](./ch12) — 複数人称標識、代名詞複数形
- [第13章](./ch13) — 否定 (`-ñ`)、コピュラの否定
- [第14章](./ch14) — 指示詞4系列、遠称 `ar` の不規則性、**第II部の総括**

## 第III部：アスペクト・修飾・分詞（第15–22章）

6相体系、形容詞の限定用法・比較級、数詞、分詞。

- [第15章](./ch15) — 終結相 (term)、impf との意味対比
- [第16章](./ch16) — 結果状態相 (result)、term との対比
- [第17章](./ch17) — 形容詞の限定用法、リンカー `-ŋ`、曲用一致
- [第18章](./ch18) — 比較級 `-jər`、出格比較、最上級 `sə + 比較級`
- [第19章](./ch19) — 起動相 (incep)・準備相 (prep)
- [第20章](./ch20) — 完遂相 (perf)、**6相体系の完成**
- [第21章](./ch21) — 基数詞、合成数詞、助数詞名詞
- [第22章](./ch22) — 能動分詞 `-ar`、時制交替、**第III部の総括**
