# Grammatica Nuçongə — Capitulum II

## 1. 対格（目的語の格）

他動詞の**直接目的語**は対格をとります。

### 1.1 対格単数

| 語幹タイプ | 語尾 | 主格 → 対格 |
|-----------|------|-------------|
| 前舌子音・有生 | **-e** | çit**i** → çit**e** |
| 前舌子音・無生 | **-e** | hin**i** → hin**e** |
| 後舌子音・有生 | **-o** | ot**a** → ot**o** |
| 後舌子音・無生 | **-o** | hñ**a** → hñ**o** |
| 前舌母音・有生 | **-je** | me**ji** → me**je** |
| 前舌母音・無生 | **-je** | ki**ji** → ki**je**、mi**ji** → mi**je** |

対格語尾は主格語尾と母音が対応します（前舌: i→e、後舌: a→o）。

**æ**（家）は不規則で、対格も **æ** です（主格と同形）。

## 2. 他動詞の活用：3人称単数・現在・非完結相

動詞は語幹に**時制母音**を付けて活用します。現在時制の3人称単数では：

| 語幹タイプ | 時制母音 | 例 |
|-----------|---------|-----|
| 前舌子音語幹 | **-e** | mir → **mire** (見る) |
| 後舌子音語幹 | **-u** | jom → **jomu** (読む) |
| | | kɔk → **kɔku** (書く) |
| 前舌子音語幹 | **-e** | tæben → **tæbene** (食べる) |

時制母音も母音調和に従います（前舌 -e、後舌 -u）。

---

## 3. SOV 語順

ヌヒョン語の基本語順は **主語(S) — 目的語(O) — 動詞(V)** です。

```
主語(主格)  目的語(対格)  動詞
sota        hño          jomu.
その男は     本を         読む。
```

定冠詞 sə は主語にも目的語にも付けられます：

> **sota** **sə hño** jomu. (その男はその本を読む。)
> **sota** hño jomu. (その男は本を読む。)

---

## 4. 主格と対格の対比

格語尾によって、同じ名詞でも文中の役割（主語か目的語か）が明確になります。

| 名詞 | 主格(主語) | 対格(目的語) |
|------|-----------|-------------|
| çit (人) | çit**i** | çit**e** |
| ot (男) | ot**a** | ot**o** |
| me (女) | me**ji** | me**je** |
| hñ (本) | hñ**a** | hñ**o** |
| ki (木) | ki**ji** | ki**je** |
| mi (実) | mi**ji** | mi**je** |

> **sota** **meje** mire. (その男は女を見る。)
> sə **meji** **oto** mire. (その女は男を見る。)

主語が主格（-a/-i/-ji）、目的語が対格（-o/-e/-je）であるため、語順が変わっても格で役割が分かります。
