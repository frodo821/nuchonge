# Grammatica Nuçongə — Capitulum XIII

## 1. 否定の仕組み

否定は接辞 **-ñ** で表します。人称標識の後に付加され、**鼻母音化規則**により時制母音が鼻母音化して -ñ 自体は脱落します。

### 1.1 鼻母音化規則

```
時制母音 + (子音*) + ñ → 鼻母音化した時制母音 + (子音*)
```

| 元の母音 | 鼻母音化 |
|---------|---------|
| e | **ẽ** |
| i | **ĩ** |
| a | **ã** |
| u | **ũ** |
| o | **õ** |
| ɔ | **ɔ̃** |
| æ | **æ̃** |
| ə | **ə̃** |

### 1.2 規則動詞の否定（jom, 現在非完結）

| | 肯定 | 否定 | 変化 |
|---|------|------|------|
| 1sg | jomuw | **jomũw** | u→ũ |
| 2sg | jomuj | **jomũj** | u→ũ |
| 3sg | jomu | **jomũ** | u→ũ |
| 1pl | jomul | **jomũl** | u→ũ |
| 2pl | jomus | **jomũs** | u→ũ |
| 3pl | jomur | **jomũr** | u→ũ |

### 1.3 前舌動詞の否定（mir, 現在非完結）

| | 肯定 | 否定 |
|---|------|------|
| 1sg | mirew | **mirẽw** |
| 2sg | miri | **mirĩ** |
| 3sg | mire | **mirẽ** |

### 1.4 過去・未来の否定

| 時制 | 肯定 3sg | 否定 3sg | 鼻母音化される母音 |
|------|---------|---------|------------------|
| 現在 | jomu | jomũ | u→ũ |
| 過去 | jomɔtɔ | jomɔ̃tɔ | 最初のɔ→ɔ̃ |
| 未来 | jomo | jomõ | o→õ |

## 2. コピュラ ar の否定

コピュラは独自の否定形を持ちます：

| | 肯定 | 否定 |
|---|------|------|
| 1sg | dərw | **də̃rw** |
| 2sg | ar | **ãr** |
| 3sg (有生) | ir | **ĩr** |
| 3sg (無生) | er | **ẽr** |
| 1pl | dərl | **də̃rl** |
| 2pl | ars | **ãrs** |
| 3pl (有生) | irr | **ĩrr** |
| 3pl (無生) | err | **ẽrr** |

> kiji **er**. kiji **ẽr**. (木がある。木がない。)
> Taka æid **ir**. Taka æid **ĩr**. (Takは家にいる。Takは家にいない。)
