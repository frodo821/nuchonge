# Grammatica Nuçongə — Capitulum VII

## 1. 所格（場所の格）

所格は**場所**を表します。「〜で」「〜に（いる/ある場所）」に相当します。

### 1.1 所格単数

| 語幹タイプ | 語尾 | 例 |
|-----------|------|-----|
| 前舌子音 | **-de** | ʃit → **ʃitde** (下で) |
| 後舌子音 | **-do** | tokl → **tokldo** (場所で) |
| 前舌母音 | **-de** | ki → **kide** (木で), uæ → **uæde** (上で) |
| 中立子音 | **-də** | — |
| æ (不規則) | — | **æid** (家で) |

> Taka **æid** ir. (Takは家にいる。)
> tolra **tokldo** irr. (鳥たちは場所にいる。)
> hña **æid** er. (本は家にある。)

人称代名詞は所格を持ちません（人は場所ではないため）。

### 1.2 空間名詞と属格の組み合わせ

空間名詞 **uæ**（上）と **ʃit**（下）は、属格名詞と組み合わせて「〜の上/下で」を表現します：

```
名詞(属格) + 空間名詞(所格)
```

> tola **kime uæde** ir. (鳥は木の上にいる。)
> sakənə **kime ʃitde** ir. (魚は木の下にいる。)
> hña **æim uæde** er. (本は家の上にある。)

## 2. 共格（共同の格）

共格は**共同動作**を表します。「〜と共に」に相当します。

### 2.1 共格単数

| 語幹タイプ | 語尾 | 例 |
|-----------|------|-----|
| 前舌子音 | **-te** | Lin → **Linte** |
| 後舌子音 | **-to** | Tak → **Takto** |
| 前舌母音 | **-te** | — |
| 中立子音 | **-tə** | — |
| æ (不規則) | — | **æit** |

代名詞の共格：

| 代名詞 | 共格sg |
|--------|--------|
| vəç | **vəçet** |
| ant | **antet** |
| kar | **kart** |

> Taka **Linte** hño jomu. (TakはLinと共に本を読む。)
> vəç **antet** kije mirew. (私はあなたと共に木を見る。)

---

## 3. 所格と共格の対比

所格 **-de/-do** は場所（どこで）、共格 **-te/-to** は同伴者（誰と）を表します：

> Taka **Linde** hño jomu. (TakはLin**のところで**本を読む。)
> Taka **Linte** hño jomu. (TakはLin**と共に**本を読む。)

## 4. 第I部の格体系総覧

第1章から第7章で6つの格を学びました：

| 格 | 機能 | 語尾(前舌) | 語尾(後舌) | 導入 |
|----|------|-----------|-----------|------|
| 主格 | 主語 | -i | -a | Ch 1 |
| 対格 | 直接目的語 | -e | -o | Ch 2 |
| 属格 | 所有 | -ne/-me | -no/-mo | Ch 3 |
| 与格 | 間接目的語 | -iñ | -uñ | Ch 6 |
| 所格 | 場所 | -de | -do | Ch 7 |
| 共格 | 共同 | -te | -to | Ch 7 |

残りの2格（**出格**と**向格**）は第II部で学びます。

### Lin の6格変化

| 格 | 形 | 例文 |
|----|-----|------|
| 主格 | **Lini** | Lini hño jomu. (Linが読む) |
| 対格 | **Line** | Taka Line mire. (TakがLinを見る) |
| 属格 | **Linne** | Taka Linne hño jomu. (Linの本を読む) |
| 与格 | **Liniñ** | Taka Liniñ hño atɔwu. (Linに与える) |
| 所格 | **Linde** | Taka Linde hño jomu. (Linの所で読む) |
| 共格 | **Linte** | Taka Linte hño jomu. (Linと共に読む) |
